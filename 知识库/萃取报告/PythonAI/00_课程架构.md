## 两大层级

### Level 1：Python编程基础与智能硬件入门（约50节）

| 阶段 | 核心目标 | 课时 |
|------|----------|------|
| PYAI 1-0 | 基础语法与逻辑启蒙 | 11节 |
| PYAI 1-1 | 函数封装与交互机制 | 11节 |
| PYAI 1-2 | 算法逻辑与数值运算 | 9节 |
| PYAI 1-3 | 数据结构与复杂逻辑 | 8节 |
| PYAI 1-4 | 智能硬件与AI应用 | 10节 |

**目标年龄**：小学中高年级（约3-5年级）

### Level 2：算法进阶与人工智能应用（约35节）

| 阶段 | 核心目标 | 课时 |

**目标年龄**：小学高年级至初中（约5-8年级）

## 软件平台

| 平台 | 用途 |
|------|------|
| 扣哒世界 (CodeCombat) | Level 1前半段语法闯关练习 |
| 斯坦星球编程+平台 | 过渡阶段及项目制作 |
| Python IDE / IDLE | Level 2及进阶代码编写 |

**核心库**
- Level 1: math, time, random
- Level 2: OpenCV (cv2), MediaPipe, NumPy, mpython (硬件库), pygame (音频)

## 硬件教具（1-4, 2-3阶段）

| 类型 | 设备 |
|------|------|
| 主控板 | NovaStar ESP32开发板 / 掌控板 |
| 传感器 | 光线传感器、MQ-2B气体传感器、土壤湿度传感器 |
| 执行器 | LED模块、RGB灯环、N20电机/风扇、舵机、水泵 |
| 其他 | USB数据线(Type-C)、杜邦线、扩展板 |

## 标准课堂流程（90分钟）

| 环节 | 时长 | 内容 |
|------|------|------|
| 课前准备 | 2-3分钟 | 检查设备、登录平台、打开课件 |
| 复习回顾 | 5-10分钟 | 提问上节课重点，检查课后作业 |
| 单词学习 | 2-3分钟 | 学习编程英语单词（variable, loop等） |
| 知识点讲解 | 10-15分钟 | 讲解新语法，结合CodeCombat闯关 |
| 项目实操 | 40-50分钟 | **核心环节**：完成当堂项目 |
| 分享总结 | 10-15分钟 | 学生上台展示，老师点评 |
| 拓展练习 | 课后/剩余时间 | 额外挑战任务 |

**底层逻辑**
"学-练-用-创"闭环，确保学生不仅掌握语法，更能应用于解决实际问题。
