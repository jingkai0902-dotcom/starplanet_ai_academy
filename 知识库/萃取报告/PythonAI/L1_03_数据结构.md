### 1-3-4 列表进阶

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程编号** | 1-3-4 |
| **课程名称** | 列表进阶 |
| **认知负荷** | 高 |
| **核心技能** | insert/pop/remove、sort()、in运算 |

#### 详细教学流程

**步骤1：知识讲解（20分钟）** `#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天学习列表的更多操作方法，让你成为列表操作大师！"
>
> "**insert()方法**：在指定位置插入元素
> ```python
> fruits = ['apple', 'cherry']
> fruits.insert(1, 'banana')  # 在索引1处插入
> print(fruits)  # 输出: ['apple', 'banana', 'cherry']
> ```
>
> **pop()方法**：按索引删除并返回元素
> ```python
> fruits = ['apple', 'banana', 'cherry']
> removed = fruits.pop(1)  # 删除索引1的元素
> print(removed)  # 输出: banana
> print(fruits)   # 输出: ['apple', 'cherry']
> ```
>
> **remove()方法**：按值删除第一个匹配元素
> ```python
> fruits = ['apple', 'banana', 'apple']
> fruits.remove('apple')  # 删除第一个apple
> print(fruits)  # 输出: ['banana', 'apple']
> ```
>
> **sort()方法**：排序列表
> ```python
> numbers = [3, 1, 4, 1, 5, 9]
> numbers.sort()
> print(numbers)  # 输出: [1, 1, 3, 4, 5, 9]
> ```
>
> **in运算符**：检查元素是否在列表中
> ```python
> fruits = ['apple', 'banana']
> print('apple' in fruits)   # 输出: True
> print('cherry' in fruits)  # 输出: False
> ```"

**步骤2：闯关实践（35分钟）** `#高负荷-产出` `#IFC-即时`

**教师话术**
> "现在来做一个成绩处理系统！输入一组成绩，排序后去掉最高分和最低分，计算平均分。"
>
> ```python
> scores = [85, 92, 78, 95, 88, 72, 90]
> scores.sort()  # 排序
> scores.pop(0)  # 去掉最低分
> scores.pop(-1) # 去掉最高分（或用pop(len(scores)-1)）
> # 计算平均分
> total = 0
> i = 0
> while i < len(scores):
>     total += scores[i]
>     i += 1
> average = total / len(scores)
> print(average)
> ```

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 混淆pop和remove | 画表格对比 | pop按位置删，remove按值删 |
| sort后找不到原顺序 | 讲解原地修改 | "sort会改变原列表，想保留原顺序用sorted()" |
| in运算不理解 | 类比现实 | "就像问'苹果在不在水果篮里？'" |

---

**核心知识点**
- insert(位置, 值)：在指定位置插入元素
- pop(索引)：按索引删除并返回元素
- remove(值)：按值删除第一个匹配元素
- sort()：原地排序列表
- in运算符：检查元素是否存在

### 1-3-7 三人成虎

#### 课程基本信息

| 项目 | 内容 |
|------|------|
| **课程编号** | 1-3-7 |
| **课程名称** | 三人成虎 |
| **认知负荷** | 高 |
| **核心技能** | 二维列表进阶、模式识别 |

#### 详细教学流程

**步骤1：知识讲解（20分钟）** `#中负荷-操练` `#IFC-即时`

**教师话术**
> "上节课我们学了二维列表表示棋盘，今天来学习如何在二维列表中**识别模式**——比如三消游戏中的'三个连在一起'！"
>
> "**横向检测**：检查同一行是否有三个连续相同
> ```python
> board = [
>     [1, 1, 1, 0, 2],
>     [0, 2, 2, 2, 0],
>     [1, 0, 1, 0, 1]
> ]
>
> # 检测第一行是否有三连
> for col in range(3):  # 只需检查到倒数第3个
>     if board[0][col] == board[0][col+1] == board[0][col+2]:
>         if board[0][col] != 0:  # 0表示空
>             print(f'发现三连！位置: 行0, 列{col}到{col+2}')
> ```
>
> **纵向检测**：检查同一列是否有三个连续相同
> ```python
> # 检测第一列是否有三连
> for row in range(len(board) - 2):
