## 详细教学流程

### 步骤1：知识讲解（15分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天是激动人心的**竞技场对战**！你要用学过的所有函数知识，编写一个AI来和对手PK！"
>
> "策略制定的关键：
> ```python
> def choose_action():
>     enemy = hero.findNearestEnemy()
>     if hero.health < 50:
>         return 'retreat'  # 血量低就撤退
>     elif enemy and hero.distanceTo(enemy) < 10:
>         return 'attack'   # 敌人近就攻击
>     else:
>         return 'advance'  # 否则前进
> ```
>
> 好的策略要考虑：血量、距离、敌人数量等多种因素！"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 策略太简单 | 引导思考 | "如果敌人很多怎么办？如果有队友呢？" |
| 代码太乱 | 模块化建议 | 把攻击、防御、移动分成不同函数 |
| 想看别人策略 | 组织分享 | 让同学互相展示和学习 |

---

### 步骤2：竞技实践（35分钟）

`#高负荷-产出` `#IFC-即时`

**教师话术**
> "现在进入红蓝对抗！你的代码就是你的军师，帮你的英雄做出最佳决策。看谁的策略更厉害！"

**禁忌提醒**
- 只攻不守——容易被反杀
- 策略单一——容易被针对

## 核心知识点

- **策略制定**：根据条件选择不同行动
- **函数综合**：组合多个函数实现复杂逻辑
- **红蓝对抗**：实时对战检验代码质量

---

## 教学类比

> 编写AI策略就像当军师，要根据战场形势随机应变。
