## 详细教学流程

### 步骤1：知识讲解（15分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天学习Python的**算术运算符**，有些你可能没见过！"
>
> "常见运算符：
> - `+` `-` `*` `/`：加减乘除
> - `//`：**整除**（只取整数部分）
> - `%`：**取余**（除法的余数）
> - `**`：**幂运算**（乘方）
>
> ```python
> print(7 // 2)   # 输出: 3（整除）
> print(7 % 2)    # 输出: 1（余数）
> print(2 ** 3)   # 输出: 8（2的3次方）
> ```
>
> 还有一个神奇的**eval()函数**，可以把字符串当代码执行：
> ```python
> result = eval('2 + 3 * 4')
> print(result)  # 输出: 14
> ```"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 混淆/和// | 对比演示 | 7/2=3.5，7//2=3 |
| 不理解取余 | 用分糖果类比 | "7颗糖分给2人，每人3颗，剩1颗" |
| 对eval好奇 | 安全提醒 | eval很强大但要小心，不要执行不信任的输入 |
