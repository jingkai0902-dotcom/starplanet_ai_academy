## 详细教学流程

**步骤1：知识讲解（20分钟）** `#中负荷-操练` `#IFC-即时`

**教师话术**
> "之前我们的函数只是执行动作，今天学习让函数**返回结果**！
>
> **return语句**：让函数把计算结果传回来
> ```python
> def add(a, b):
>     result = a + b
>     return result  # 返回计算结果
>
> total = add(3, 5)  # total = 8
> ```
>
> **利息计算模型**：
> ```python
> def calculate_interest(principal, rate, years):
>     # 复利公式：本金 × (1 + 利率)^年数
>     amount = principal * (1 + rate) ** years
>     interest = amount - principal
>     return interest
>
> # 存1000元，年利率5%，存3年
> profit = calculate_interest(1000, 0.05, 3)
> print(f'利息：{profit:.2f}元')  # 利息：157.63元
> ```"

**步骤2：闯关实践（35分钟）** `#高负荷-产出` `#IFC-即时`

**教师话术**
> "现在你是银行程序员！设计一个存取款系统：
> 1. 存款函数：返回存款后的余额
> 2. 取款函数：检查余额是否足够，返回取款后的余额
> 3. 利息计算：返回指定年限后的利息"
