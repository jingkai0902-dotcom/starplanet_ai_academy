## 详细教学流程

**步骤1：知识讲解（20分钟）** `#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天学习**数字位分离**，这是解密坐标的关键技能！
>
> **幂运算(**)和整除(//)：
> ```python
> # 幂运算：计算次方
> print(2 ** 3)    # 8（2的3次方）
> print(10 ** 2)   # 100（10的2次方）
>
> # 整除：只取整数部分
> print(17 // 5)   # 3（17÷5=3余2，只取3）
> print(123 // 10) # 12（去掉个位）
> ```
>
> **数字位分离技巧**：
> ```python
> num = 1234
> # 取个位：对10取余
> ones = num % 10      # 4
> # 取十位：先整除10，再对10取余
> tens = (num // 10) % 10   # 3
> # 取百位：先整除100，再对10取余
> hundreds = (num // 100) % 10  # 2
> # 取千位：整除1000
> thousands = num // 1000  # 1
> ```"

**步骤2：闯关实践（35分钟）** `#高负荷-产出` `#IFC-即时`

**教师话术**
> "你收到一个加密坐标：5738。需要解密：
> - 千位和百位组成X坐标：57
> - 十位和个位组成Y坐标：38
> 用数字位分离技术解密！"

## 核心知识点

- 幂运算(**)：计算次方，如 `2**3=8`
- 整除(//)：只取整数部分，如 `17//5=3`
- 数字位分离：`%10`取个位，`//10`去个位
