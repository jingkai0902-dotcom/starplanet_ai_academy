## 详细教学流程

**步骤1：知识讲解（20分钟）** `#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天我们学习两个重要的逻辑运算符：
>
> **逻辑与(and)**：两个条件都满足才为True
> ```python
> if health > 0 and has_weapon:
>     attack()
> ```
>
> **逻辑或(or)**：满足其一即为True
> ```python
> if is_enemy or is_trap:
>     avoid()
> ```
>
> 还有**break语句**：跳出循环
> ```python
> while True:
>     if victory:
>         break  # 跳出循环
> ```"

## 核心知识点

- 逻辑与(and)：两个条件都满足才为True
- 逻辑或(or)：满足其一即为True
- break语句：跳出死循环
