## 详细教学流程

### 步骤1：知识讲解（15分钟）
`#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天我们要用Python画一个炫彩风车！还记得Scratch里的画笔功能吗？Python也有类似的工具，叫做turtle（海龟）。
>
> **turtle基础命令**：
> ```python
> import turtle        # 导入海龟模块
> t = turtle.Turtle()  # 创建一只海龟
> t.forward(100)       # 向前走100步
> t.right(90)          # 右转90度
> t.left(45)           # 左转45度
> t.penup()            # 抬起画笔（移动不画线）
> t.pendown()          # 放下画笔（移动画线）
> t.color('red')       # 设置颜色
> ```
>
> **循环画图**：
> ```python
> for i in range(4):   # 重复4次
>     t.forward(100)   # 向前100步
>     t.right(90)      # 右转90度
> # 这样就画出一个正方形！
> ```"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 不理解for循环 | 对比while循环 | "for是知道次数的循环，while是不知道次数的" |
| 角度计算困难 | 提供公式 | "正N边形的外角 = 360 / N" |
| 想画更复杂图形 | 引入嵌套循环 | 用两层循环画螺旋图案 |

### 步骤2：闯关实践（35分钟）
`#高负荷-产出` `#IFC-即时`

**教师话术**
> "现在我们来画风车！
>
> 任务清单：
> 1. 画一个风车叶片（三角形或扇形）
> 2. 用循环画4个叶片，每个旋转90度
> 3. 给每个叶片设置不同颜色
> 4. 让风车转起来（动画效果）
>
> 提示：画完一个叶片后，回到中心点，转90度，再画下一个！"

**即时反馈检查点**
- 检查import语句是否正确
- 检查for循环的range参数
- 检查角度计算是否正确

**核心代码模式**
```python
import turtle

t = turtle.Turtle()
t.speed(0)  # 最快速度

colors = ['red', 'blue', 'green', 'yellow']

for i in range(4):
    t.color(colors[i])
    t.begin_fill()
    for j in range(3):
        t.forward(100)
        t.left(120)
    t.end_fill()
    t.right(90)

turtle.done()
```

**禁忌提醒**
- 不要忘记`import turtle`——会报NameError
- 不要混淆`left`和`right`——方向会反
- 不要忘记`turtle.done()`——窗口会立即关闭
