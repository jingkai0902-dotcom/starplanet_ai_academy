## 详细教学流程

### 步骤1：知识讲解（15分钟）
`#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天我们学三个重要的知识点：
>
> **第一，注释**：用 `#` 开头的文字，电脑会跳过不执行。注释是写给人看的，帮助我们理解代码。
> ```python
> # 这是一条注释，电脑不会执行
> hero.moveRight()  # 这也是注释，写在代码后面
> ```
>
> **第二，无限循环**：`while True:` 表示一直重复执行。
> ```python
> while True:
>     hero.moveRight()  # 英雄会一直往右走
> ```
>
> **第三，缩进**：Python用缩进表示代码的层级关系。while下面的代码要缩进4个空格！"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 缩进总是出错 | 强调Tab键 | "每次缩进按一下Tab键，不要用空格" |
| 不理解无限循环 | 用游戏主循环类比 | "游戏画面一直在刷新，就是无限循环" |
| 忘记写冒号 | 口诀记忆 | "while True冒号，缩进不能少" |

**禁忌提醒**
- 混用空格和Tab——会导致`IndentationError`
- 忘记冒号——`while True` 后面必须加冒号
- 无限循环里没有暂停——会导致程序卡死

## 核心知识点

- 注释：`#` 符号，不影响运行但有助于理解
- 无限循环：`while True:` 让角色持续行动
- 缩进规则：4个空格或Tab，Python对缩进极度敏感

**底层逻辑**
- **代码规范**：养成写注释的好习惯
- **循环思维**：理解程序的持续运行机制
