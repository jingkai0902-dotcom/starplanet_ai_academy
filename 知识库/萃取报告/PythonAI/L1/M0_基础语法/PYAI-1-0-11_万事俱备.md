## 详细教学流程

### 步骤1：知识讲解（15分钟）
`#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天是本单元的最后一课，我们来学习比较运算符，并复习所有知识点！
>
> **比较运算符**：
> ```python
> >   大于        5 > 3   → True
> <   小于        5 < 3   → False
> ==  等于        5 == 5  → True
> !=  不等于      5 != 3  → True
> >=  大于等于    5 >= 5  → True
> <=  小于等于    3 <= 5  → True
> ```
>
> **对象属性**：
> ```python
> enemy.health    # 敌人的血量
> enemy.pos.x     # 敌人的x坐标
> enemy.pos.y     # 敌人的y坐标
> enemy.type      # 敌人的类型
> hero.health     # 英雄的血量
> hero.pos        # 英雄的位置
> ```
>
> **综合应用**：
> ```python
> if enemy.health < 20:
>     hero.attack(enemy)  # 血量低于20就攻击
> if hero.health < 50:
>     hero.shield()       # 自己血量低就防御
> ```"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 混淆`=`和`==` | 反复强调 | "一个等号是赋值，两个等号是比较" |
| 属性访问出错 | 用print调试 | 先print(enemy)看有哪些属性 |
| 比较逻辑错误 | 画数轴 | 在数轴上标出比较范围 |

### 步骤2：闯关实践（35分钟）

> 2. 用对象属性获取位置信息
> 3. 用if-elif-else做出决策
> 4. 用while循环持续保护
>
> 这是本单元的终极挑战！"

**即时反馈检查点**
- 检查比较运算符使用是否正确
- 检查属性访问是否正确（注意拼写）
- 检查整体逻辑是否完整

**核心代码模式**
```python
while True:
    # 找最近的敌人
    enemy = hero.findNearestEnemy()

    if enemy:
        # 计算与敌人的距离
        distance = hero.distanceTo(enemy)

        # 根据距离和血量决定策略
        if distance < 10:
            # 敌人很近
            if enemy.health < 30:
                hero.attack(enemy)  # 血少直接打
            else:
                hero.shield()       # 血多先防御
        elif distance < 30:
            # 敌人中等距离
            hero.attack(enemy)
        else:
            # 敌人较远，移动靠近
            hero.moveXY(enemy.pos.x, enemy.pos.y)
    else:
        # 没有敌人，回到保护位置
        hero.moveXY(40, 30)
```

**禁忌提醒**
- 比较时用`=`而不是`==`——`if health = 50`会报错
- 属性名拼写错误——`enemy.helth`会报AttributeError
- 忘记处理None情况——`enemy.health`在enemy为None时会报错
