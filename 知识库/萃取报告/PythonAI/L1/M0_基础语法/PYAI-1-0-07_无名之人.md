## 详细教学流程

### 步骤1：知识讲解（15分钟）
`#中负荷-操练` `#IFC-即时`

**教师话术**
> "今天我们学习Python中非常重要的概念——变量！
>
> **什么是变量？**
> 变量就像一个盒子，可以装东西。盒子有名字，里面装的东西可以变。
>
> ```python
> name = 'Hero'      # 创建一个变量name，装入字符串'Hero'
> health = 100       # 创建一个变量health，装入数字100
> ```
>
> **字符串**：用引号括起来的文字
> ```python
> message = 'Hello, World!'  # 单引号
> greeting = "你好！"         # 双引号也可以
> ```
>
> **print调试**：想知道变量里装了什么？用print打印出来！
> ```python
> enemy = hero.findNearestEnemy()
> print(enemy)       # 打印敌人信息
> print(enemy.pos)   # 打印敌人位置
> ```"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 变量命名困难 | 提供命名规则 | "用英文单词，见名知意，如enemy_count" |
| 混淆字符串和变量 | 强调引号 | "有引号是字符串，没引号是变量" |
| print输出看不懂 | 加标签打印 | `print('敌人位置:', enemy.pos)` |

### 步骤2：闯关实践（35分钟）
`#高负荷-产出` `#IFC-即时`

**教师话术**
> "现在进入Kithgard之门关卡！
>
> 这一关你需要：
> 1. 用变量存储找到的敌人
> 2. 用print打印敌人信息，观察敌人属性
> 3. 根据敌人位置决定攻击策略
>
> 记住：print是你的好帮手！不确定的时候就print出来看看！"

**即时反馈检查点**
- 检查变量命名是否合法（不能数字开头）
- 检查字符串引号是否成对
- 检查print语句是否正确

**核心代码模式**
```python
# 用变量存储敌人
enemy = hero.findNearestEnemy()

# 用print调试，查看敌人信息
print('找到敌人:', enemy)
print('敌人类型:', enemy.type)
print('敌人位置:', enemy.pos)

# 根据信息决定行动
if enemy:
    hero.attack(enemy)
```

**禁忌提醒**
- 变量名不能以数字开头——`1enemy`会报错，应该用`enemy1`
- 变量名不能用Python关键字——`if = 5`会报错
- 字符串引号不能混用——`'Hello"`会报错
- 中文变量名虽然可以但不推荐——可能导致编码问题

## 核心知识点

- 变量：存储数据的容器，`变量名 = 值`
- 字符串：用引号括起来的文本
- print调试：输出变量值帮助理解程序状态

**底层逻辑**
- **调试思维**：培养用print排查问题的习惯
- **抽象能力**：理解变量作为数据容器的概念
