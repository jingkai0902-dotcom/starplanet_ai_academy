## 详细教学流程

### 步骤1：知识讲解（20分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**

> "如果我想表示一个棋盘，一维列表够用吗？不够！我们需要**二维列表**——列表里面套列表。"
>
> "**二维列表**：
> ```python
> board = [
>     [1, 0, 0],
>     [0, 1, 0],
>     [0, 0, 1]
> ]
> ```
>
> **访问元素**：
> ```python
> print(board[0][0])  # 第1行第1列，输出: 1
> print(board[1][2])  # 第2行第3列，输出: 0
> ```
>
> **嵌套循环遍历**：外层遍历行，内层遍历列
> ```python
> for row in range(3):
>     for col in range(3):
>         print(board[row][col])
> ```"

## 核心知识点

- **二维列表**：列表嵌套列表 `[[1,0], [0,1]]`
- **嵌套循环**：外层遍历行，内层遍历列
- **矩阵坐标**：根据值在对应坐标绘制棋子
