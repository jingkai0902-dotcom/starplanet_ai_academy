## 详细教学流程

### 步骤1：知识讲解（20分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**

> "列表用数字索引，但有时候我们想用名字来找东西，这就需要**字典(Dict)**！"
>
> "**字典定义**：
> ```python
> student = {
>     'name': 'Tom',
>     'age': 10,
>     'grade': 'A'
> }
> ```
>
> **访问元素**：用键(key)来获取值(value)
> ```python
> print(student['name'])  # 输出: Tom
> print(student['age'])   # 输出: 10
> ```
>
> **添加/修改**：
> ```python
> student['hobby'] = 'coding'  # 添加新键值对
> student['age'] = 11          # 修改已有的值
> ```"

## 核心知识点

- **字典定义**：`{'key': 'value'}`
- **键值对**：用键来获取值
- **映射关系**：适合存储"名称-数据"对应关系
