## 详细教学流程

### 步骤1：知识讲解（20分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**

> "while循环很好用，但Python有更简洁的循环方式——**for循环**！配合**range()**函数，代码更简洁。"
>
> "**range()函数**：生成数字序列
> ```python
> range(5)      # 生成 0, 1, 2, 3, 4
> range(1, 6)   # 生成 1, 2, 3, 4, 5
> range(0, 10, 2)  # 生成 0, 2, 4, 6, 8（步长为2）
> ```
>
> **for循环**：遍历序列中的每个元素
> ```python
> for i in range(5):
>     print(i)  # 输出 0, 1, 2, 3, 4
> ```
>
> **对比while循环**：
> ```python
> # while写法（4行）
> i = 0
> while i < 5:
>     print(i)
>     i += 1
>
> # for写法（2行）
> for i in range(5):
>     print(i)
> ```
>
> **遍历列表**：
> ```python
> fruits = ['apple', 'banana', 'cherry']
> for fruit in fruits:
>     print(fruit)
> ```"

## 核心知识点

- **range(n)**：生成0到n-1的序列
- **range(start, end)**：生成start到end-1的序列
- **range(start, end, step)**：带步长的序列
- **for循环**：比while更简洁的遍历方式
