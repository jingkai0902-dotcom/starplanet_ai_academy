## 详细教学流程

### 步骤1：知识讲解（20分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**

> "之前我们遍历列表时，循环次数是写死的。如果列表长度变了怎么办？用**len()**函数！"
>
> "**len()函数**：获取列表长度
> ```python
> fruits = ['apple', 'banana', 'cherry']
> print(len(fruits))  # 输出: 3
> ```
>
> **通用遍历模式**：不管列表多长都能遍历
> ```python
> i = 0
> while i < len(fruits):
>     print(fruits[i])
>     i += 1
> ```
>
> **修改元素**：通过索引直接赋值
> ```python
> fruits[1] = 'orange'  # 把banana改成orange
> print(fruits)  # 输出: ['apple', 'orange', 'cherry']
> ```"

---

### 步骤2：闯关实践（35分钟）

`#高负荷-产出` `#IFC-即时`

**教师话术**

> "现在来玩'移花接木'！把列表中所有的负数变成0。"
>
> ```python
> numbers = [5, -3, 8, -1, 2, -7]
> i = 0
> while i < len(numbers):
>     if numbers[i] < 0:
>         numbers[i] = 0  # 移花接木！
>     i += 1
> print(numbers)  # 输出: [5, 0, 8, 0, 2, 0]
> ```

## 核心知识点

- **len()函数**：获取列表长度，返回整数
- **通用遍历**：`while i < len(list)` 适用于任意长度列表
- **元素修改**：`list[i] = new_value` 直接修改指定位置
