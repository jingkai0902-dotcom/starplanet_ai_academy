## 详细教学流程

### 步骤1：知识讲解（20分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**

> "上节课我们学了列表，但列表是固定的。如果我想往列表里加东西怎么办？用**append()**方法！"
>
> "**append()方法**：在列表末尾添加元素
> ```python
> fruits = ['apple', 'banana']
> fruits.append('cherry')
> print(fruits)  # 输出: ['apple', 'banana', 'cherry']
> ```
>
> **动态列表**：从空列表开始，逐个添加
> ```python
> numbers = []  # 空列表
> numbers.append(1)
> numbers.append(2)
> numbers.append(3)
> print(numbers)  # 输出: [1, 2, 3]
> ```
>
> **结合循环添加**：
> ```python
> squares = []
> i = 1
> while i <= 5:
>     squares.append(i * i)
>     i += 1
> print(squares)  # 输出: [1, 4, 9, 16, 25]
> ```"

---

### 步骤2：闯关实践（35分钟）

`#高负荷-产出` `#IFC-即时`

**教师话术**

> "现在我们来做一个'找不同'游戏！给你两个列表，找出第二个列表中有但第一个列表中没有的元素。"
>
> ```python
> list1 = [1, 2, 3, 4, 5]
> list2 = [1, 2, 6, 4, 7]
> different = []
> i = 0
> while i < 5:
>     if list2[i] != list1[i]:
>         different.append(list2[i])
>     i += 1
> print(different)  # 输出: [6, 7]
> ```
