## 详细教学流程

**步骤1：知识讲解（15分钟）** `#中负荷-操练` `#IFC-即时`

**教师话术**
> "同学们，之前我们写的Python代码都是在电脑上运行的。今天开始，我们要让代码控制真实的硬件！这就是**物联网(IoT)**的魅力——用代码控制物理世界。"
>
> "首先认识我们的主角——**ESP32开发板**！
> - 一块小型计算机，可以运行MicroPython代码
> - 内置WiFi和蓝牙，可以联网
> - 有很多**引脚(Pin)**，用来连接传感器和执行器
>
> **引脚类型**：
> ```
> VCC (3.3V) - 电源正极，给传感器供电
> GND        - 电源负极，接地
> P0-P20     - 数字/模拟引脚，连接传感器
> ```"

## 核心代码模式

```python
# MicroPython基础导入
from machine import Pin
import time

# 创建LED对象（假设连接到Pin 2）
led = Pin(2, Pin.OUT)

# 点亮LED
led.value(1)  # 或 led.on()

# 熄灭LED
led.value(0)  # 或 led.off()

# LED闪烁
while True:
    led.value(1)
    time.sleep(0.5)  # 亮0.5秒
    led.value(0)
    time.sleep(0.5)  # 灭0.5秒
```

## 变体示例

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 不理解引脚 | 用插座类比 | "引脚就像插座，传感器插上去就能用" |
| 混淆VCC和GND | 用水流类比 | "VCC是水龙头，GND是下水道" |
| 代码上传失败 | 检查USB连接 | 确认选择了正确的串口 |

## 禁忌提醒

- **带电接线**——会短路烧毁开发板
- **VCC直接接GND**——这是短路，绝对禁止
- **开发板放金属表面通电**——底部引脚会短路
