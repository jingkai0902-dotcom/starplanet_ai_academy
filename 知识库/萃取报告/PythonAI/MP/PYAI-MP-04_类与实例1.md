## 详细教学流程

**教师话术**
> "今天我们学习一个超级重要的概念——**面向对象编程(OOP)**！这是为了让你们理解为什么后面要写 `sensor = DHT11(pin)` 这样的代码。"
>
> "**类(Class)和实例(Instance)**：
> - 类是**模板/蓝图/图纸**，定义了一类事物的共同特征
> - 实例是根据类创建的**具体对象/实物**
>
> 比喻：类就像'做饼干的模具'，实例就是用模具压出来的'饼干'。"

---

## 核心代码模式

```python
# 定义一个"狗"类
class Dog:
    # 属性（特征）
    name = '小白'
    color = '白色'

    # 方法（行为）
    def bark(self):
        print('汪汪！')

# 创建实例
dog1 = Dog()
dog1.name = '旺财'
dog1.bark()  # 输出：汪汪！

dog2 = Dog()
dog2.name = '小黑'
print(dog2.name)  # 输出：小黑
```
