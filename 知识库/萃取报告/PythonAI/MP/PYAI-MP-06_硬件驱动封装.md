## 详细教学流程

**教师话术**
> "前两节课我们学了类和实例。现在来看看**NovaMP部件库**——它就是把各种传感器的驱动封装成了类！"
>
> "**三种import方式**：
> ```python
> # 方式1：导入整个模块
> import novaMP
> led = novaMP.LED(pin)
>
> # 方式2：从模块导入特定类
> from novaMP import LED
> led = LED(pin)
>
> # 方式3：导入所有（不推荐）
> from novaMP import *
> ```
>
> **标准化接口**：所有NovaMP驱动都有统一的方法
> - `help()` - 查看使用说明
> - `test()` - 测试硬件是否正常"

---

## 核心代码模式

```python
# 导入NovaMP部件库
from novaMP import LED, Button

# 创建LED实例
led = LED(2)

# 查看帮助
led.help()  # 显示LED的使用说明

# 测试硬件
led.test()  # LED会闪烁几下，确认硬件正常

# 正常使用
led.fill((255, 0, 0))  # 设置红色
led.write()            # 更新显示
```
