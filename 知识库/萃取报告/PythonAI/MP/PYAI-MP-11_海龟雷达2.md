## 详细教学流程

**教师话术**
> "**串口通信(UART)**：让ESP32和电脑'对话'
> - TX(发送) 接 RX(接收)
> - RX(接收) 接 TX(发送)
> - 必须**交叉连接**！"

---

## 核心代码模式

```python
from machine import UART
import time

# 创建串口实例
uart = UART(2, baudrate=9600, tx=17, rx=16)

# 发送数据到电脑
uart.write('Hello from ESP32!\n')

# 接收电脑发来的数据
if uart.any():
    data = uart.read()
    print(f"收到: {data}")
```

---

## 串口连接示意图

```
ESP32                    电脑/USB转串口
  TX  ------------------>  RX
  RX  <------------------  TX
  GND ------------------>  GND
```

**重点**：TX和RX必须交叉连接！

---

## 禁忌提醒

- **TX接TX，RX接RX**——通信失败，必须交叉连接
- **串口被占用时烧录代码**——需要先停止程序或重刷固件
