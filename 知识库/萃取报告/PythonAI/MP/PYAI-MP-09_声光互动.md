## 详细教学流程

**教师话术**
> "今天做一个**音强流水灯**——灯光随着声音的强弱变化！"
>
> "**RGB三基色**：
> - R(红)、G(绿)、B(蓝)，每个值0-255
> - (255, 0, 0) = 红色
> - (0, 255, 0) = 绿色
> - (255, 255, 0) = 黄色（红+绿）
>
> **NeoPixel灯带**：可以单独控制每颗LED的颜色"

---

## 核心代码模式

```python
from novaMP import NeoPixel, Microphone
import time

# 创建实例
np = NeoPixel(4, 12)  # Pin 4, 12颗灯珠
mic = Microphone(34)

while True:
    # 读取音量
    sound = mic.read()

    # 根据音量调整亮度
    brightness = min(255, sound // 16)

    # 设置所有灯珠颜色
    for i in range(12):
        np[i] = (brightness, 0, brightness)  # 紫色

    np.write()  # 更新显示
    time.sleep_ms(50)
```
