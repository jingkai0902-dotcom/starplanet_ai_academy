## 详细教学流程

**教师话术**
> "**超声波测距原理**：
> 1. 发射超声波
> 2. 遇到障碍物反射回来
> 3. 计算时间差，算出距离
> - 距离 = 时间 x 声速 / 2"

---

## 核心代码模式

```python
from novaMP import HCSR04
import time

# 创建超声波传感器实例
sonar = HCSR04(trig=5, echo=18)

while True:
    distance = sonar.distance_cm()
    print(f"距离: {distance} cm")
    time.sleep_ms(100)
```

---

## 超声波测距原理图解

```
发射器 -----> 超声波 -----> 障碍物
                              |
                              v
接收器 <----- 反射波 <----- 反射
```

**计算公式**：
- 声速 = 340 m/s = 0.034 cm/us
- 距离 = 时间(us) x 0.034 / 2
