>
> "**圆形碰撞公式**：当两个圆心的距离 ≤ 两个半径之和时，判定为碰撞。
>
> ```python
> import math
>
> def is_collision(x1, y1, r1, x2, y2, r2):
>     distance = math.sqrt((x2-x1)**2 + (y2-y1)**2)

| 想简化代码 | 介绍distanceTo | CodeCombat内置的距离函数 |

---

**核心知识点**
- 圆形碰撞公式：当两个圆心距离 ≤ R+r 时判定为碰撞
- 运算优先级：括号 > 乘方 > 乘除 > 加减

**步骤1：知识讲解（20分钟）** `#中负荷-操练` `#IFC-即时`

**教师话术**
> "上节课我们学了圆形碰撞，今天来学**矩形碰撞检测**！
>
> 想象两个盒子，什么时候它们会碰撞？当它们在X轴和Y轴上都有重叠时！
>
> **矩形碰撞公式**（AABB检测）：
> ```python
> def rect_collision(x1, y1, w1, h1, x2, y2, w2, h2):
>     # 检查X轴是否重叠
>     x_overlap = x1 < x2 + w2 and x1 + w1 > x2
>     # 检查Y轴是否重叠
>     y_overlap = y1 < y2 + h2 and y1 + h1 > y2
>     # 两个轴都重叠才算碰撞
>     return x_overlap and y_overlap
> ```
>
> **比较运算符优先级**：`<`、`>`、`<=`、`>=` 优先级相同，从左到右计算。"

**步骤2：闯关实践（35分钟）** `#高负荷-产出` `#IFC-即时`

**教师话术**

>
> 提示：把车辆和禁行区域都看作矩形，用碰撞检测判断。"

**变体示例**

**教师话术**
> "今天学习**数字位分离**，这是解密坐标的关键技能！
>
> **幂运算(**)和整除(//)：
> ```python
> # 幂运算：计算次方
> print(2 ** 3)    # 8（2的3次方）
> print(10 ** 2)   # 100（10的2次方）
>
> # 整除：只取整数部分
> print(17 // 5)   # 3（17÷5=3余2，只取3）
> print(123 // 10) # 12（去掉个位）
> ```
>
> **数字位分离技巧**：
> ```python
> num = 1234
> # 取个位：对10取余
> ones = num % 10      # 4
> # 取十位：先整除10，再对10取余
> tens = (num // 10) % 10   # 3
> # 取百位：先整除100，再对10取余
> hundreds = (num // 100) % 10  # 2
> # 取千位：整除1000
> thousands = num // 1000  # 1
> ```"

**步骤2：闯关实践（35分钟）** `#高负荷-产出` `#IFC-即时`

**教师话术**
> "你收到一个加密坐标：5738。需要解密：
> - 千位和百位组成X坐标：57
> - 十位和个位组成Y坐标：38
> 用数字位分离技术解密！"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 混淆//和/ | 对比演示 | `7/2=3.5`，`7//2=3` |
| 取位顺序错 | 画数位图 | 标出千、百、十、个位 |
| 负数取余 | 暂时回避 | 负数取余规则复杂，先用正数 |

---

**核心知识点**
- 幂运算(**)：计算次方，如 `2**3=8`
- 整除(//)：只取整数部分，如 `17//5=3`
- 数字位分离：`%10`取个位，`//10`去个位
