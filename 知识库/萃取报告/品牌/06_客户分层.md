## 一、客户分层模型

### 1.1 四类核心客户画像

| 客户类型 | 画像特征 | 核心痛点 | 决策特点 | 沟通策略 |
|----------|----------|----------|----------|----------|
| **A类：小白家长** | 对STEM了解少，关注短期效果 | 写作业慢、字迹差、好动 | 感性决策，看效果 | 单点突破，具象化 |
| **B类：焦虑家长** | 教育焦虑重，怕孩子落后 | 升学压力、竞争焦虑 | 从众心理强 | 社会认同，权威背书 |
| **C类：理性家长** | 高知背景，逻辑性强 | 质疑效果，对比竞品 | 理性分析，货比三家 | 数据说话，价值锚定 |
| **D类：老客户** | 信任机构，关注软实力 | 性格培养、习惯养成 | 情感决策，重关系 | 情感共鸣，承诺一致 |

**底层逻辑**：
"因材施教"销售策略——不同认知水平的家长需要不同的沟通方式。对低认知家长讲太多理论会适得其反，对高知家长讲太浅会显得不专业。

## 二、A类客户：小白家长

### 2.1 客户画像

**典型特征**：
- 对STEM/编程了解很少
- 关注短期、可见的效果
- 决策依赖直觉和感受
- 容易被具体案例打动

**常见表现**：
- "学编程有什么用？"
- "能提高成绩吗？"
- "孩子写作业太慢了"
- "别的孩子都在学什么？"

### 2.2 沟通策略：单点突破

**核心原则**：只解决一个问题，不讲大道理

**标准话术**：
> "妈妈，您说孩子写作业慢，其实这是手眼协调能力的问题。我们的课程通过键盘鼠标操作，专门训练这个能力。练好了，写作业自然就快了。"
> `#具象化` `#缓解焦虑` [UID: SL-SEG-A01]

**变体示例**：

| 痛点 | 话术 | 心理学标签 |
|------|------|------------|
| 写作业慢 | "手眼协调练好了，写字速度能快一倍。" | `#具象化` |
| 字迹潦草 | "精细拼搭锻炼小肌肉，字自然写得工整。" | `#具象化` |
| 坐不住 | "编程需要专注，孩子的注意力会慢慢提升。" | `#具象化` |
| 不爱动脑 | "游戏化学习，孩子玩着玩着就爱上思考了。" | `#缓解焦虑` |

### 2.3 禁忌提醒

| 错误做法 | 正确做法 | 原因 |
|----------|----------|------|
| 讲STEM教育理念 | 讲具体效果 | 小白家长听不懂理念 |
| 讲课程体系架构 | 讲孩子能学会什么 | 太抽象无法理解 |
| 讲多个好处 | 只讲一个痛点 | 信息过载反而犹豫 |
| 用专业术语 | 用生活化语言 | 术语造成距离感 |

### 2.4 成交关键

**促单话术**：
> "您可以先让孩子试听一节，看看他喜不喜欢。试听是免费的，没有任何压力。"
> `#缓解焦虑` `#互惠原则` [UID: SL-SEG-A02]

**成交信号**：
- 问"多少钱"
- 问"什么时候上课"
- 让孩子试玩教具
- 主动加微信

## 三、B类客户：焦虑家长

### 3.1 客户画像

**典型特征**：
- 教育焦虑感强烈
- 担心孩子落后于人
- 关注升学、竞赛出口
- 容易被"别人都在学"打动

**常见表现**：
- "现在孩子竞争太激烈了"
- "别的孩子都在学，我们不能落后"
- "这个对升学有帮助吗？"
- "能参加什么比赛？"

### 3.2 沟通策略：社会认同+权威背书

**核心原则**：利用从众心理，强调权威出口

**标准话术**：
> "现在很多家长都在让孩子学编程，因为这是未来的必备技能。我们的课程对标信息学奥赛，很多学员都拿到了省级奖项，这对升学是很有帮助的。"
> `#社会认同` `#权威背书` [UID: SL-SEG-B01]

**变体示例**：

| 场景 | 话术 | 心理学标签 |
|------|------|------------|
| 担心落后 | "我们这个班已经有8个孩子了，都是附近小区的。" | `#社会认同` |
| 关注升学 | "编程是科技特长生的重要路径，很多名校都认可。" | `#权威背书` |
| 关注竞赛 | "我们的学员在FLL、VEX机器人赛中获得XX个奖项。" | `#权威背书` |
| 怕错过 | "3-6岁是思维发展的黄金期，错过了补不回来。" | `#损失规避` |

### 3.3 禁忌提醒

| 错误做法 | 正确做法 | 原因 |
|----------|----------|------|
| 否定焦虑 | 认同焦虑 | 焦虑是真实的，否定会失去信任 |
| 过度承诺 | 如实说明 | 夸大效果会导致后期投诉 |
| 制造更多焦虑 | 提供解决方案 | 焦虑过度会导致决策瘫痪 |
| 贬低其他选择 | 突出自身优势 | 贬低会显得不专业 |

### 3.4 成交关键

**促单话术**：
> "这个班只剩3个名额了，下一期要等一个月。很多家长都是先占个位置，您要不要也先报上？"
> `#稀缺性建立` `#社会认同` [UID: SL-SEG-B02]

**成交信号**：
- 问"别的孩子都学什么"
- 问"能参加什么比赛"
- 问"对升学有帮助吗"
- 表现出紧迫感

## 四、C类客户：理性家长

### 4.1 客户画像

**典型特征**：
- 高知背景，逻辑性强
- 会货比三家，理性分析
- 质疑效果，需要数据支撑
- 可能迷信奥数，轻视编程

**常见表现**：
- "你们和XX机构有什么区别？"
- "学编程对数学有帮助吗？"
- "有什么数据证明效果？"
- "奥数不是更重要吗？"

### 4.2 沟通策略：价值锚定+对比效应

**核心原则**：用数据和逻辑说话，承认对方观点再引导

**标准话术**：
> "您说得对，奥数确实很重要，它训练的是数学技巧。但编程培养的是计算思维——分解问题、逻辑推理、系统思考。这是'道'和'术'的区别。奥数是术，编程是道。道练好了，术自然就通了。"
> `#价值锚定` `#对比效应` [UID: SL-SEG-C01]

**变体示例**：

| 质疑 | 话术 | 心理学标签 |
|------|------|------------|
| 和竞品对比 | "很多机构只教编程，我们是软硬结合，培养综合能力。" | `#对比效应` |
| 质疑效果 | "我们的学员平均专注力提升40%，这是有数据支撑的。" | `#权威背书` |
| 奥数更重要 | "奥数是术，编程是道。道练好了，术自然就通了。" | `#价值锚定` |
| 太贵了 | "一套完整的课程体系，培养的是终身受用的能力。" | `#价值锚定` |

### 4.3 禁忌提醒

| 错误做法 | 正确做法 | 原因 |
|----------|----------|------|
| 否定奥数 | 承认奥数价值 | 否定会激起对抗 |
| 空洞承诺 | 数据支撑 | 理性家长需要证据 |
| 情感攻势 | 逻辑说服 | 情感对理性家长无效 |
| 回避对比 | 正面对比 | 回避会显得心虚 |

### 4.4 成交关键

**促单话术**：
> "您可以都试听一下，对比后再决定。我相信您会看到差别的。我们的课程体系从3岁到16岁全覆盖，对标权威竞赛，这是很多机构做不到的。"
> `#对比效应` `#权威背书` [UID: SL-SEG-C02]

**成交信号**：
- 详细询问课程体系
- 要求看学员作品
- 对比多家后回来
- 认可课程理念

## 五、D类客户：老客户

### 5.1 客户画像

**典型特征**：
- 已经是学员家长
- 信任机构和老师
- 关注孩子软实力
- 情感决策，重关系

**常见表现**：
- "孩子很喜欢这里"
- "老师对孩子很好"
- "想让孩子继续学"
- "有什么新课程吗？"

### 5.2 沟通策略：情感共鸣+承诺一致

**核心原则**：强化情感连接，利用已有承诺

**标准话术**：
> "XX妈妈，孩子这段时间进步很大！您之前说最希望他能坐得住、学得进，现在他已经能专注完成一个完整项目了。下一阶段会学习更有挑战性的内容，建议不要中断。"
> `#情感共鸣` `#承诺一致` [UID: SL-SEG-D01]

**变体示例**：

| 场景 | 话术 | 心理学标签 |
|------|------|------------|
| 续费沟通 | "孩子好不容易建立的习惯，中断了太可惜。" | `#损失规避` |
| 升班推荐 | "孩子已经准备好了，下一阶段会更有挑战性。" | `#承诺一致` |
| 转介绍 | "您推荐朋友来，双方都能获得赠课。" | `#互惠原则` |
| 活动邀请 | "我们有个学员展示活动，XX表现这么好，一定要来。" | `#社会认同` |

### 5.3 禁忌提醒

| 错误做法 | 正确做法 | 原因 |
|----------|----------|------|
| 只谈续费 | 先谈孩子进步 | 功利感太强会伤害关系 |
| 忽视情感 | 强化情感连接 | 老客户最看重关系 |
| 推销新课 | 根据需求推荐 | 硬推会降低信任 |
| 忘记承诺 | 回顾之前的期望 | 承诺一致增加续费率 |

### 5.4 成交关键

**促单话术**：
> "老学员续费有专属优惠，而且可以继续跟着XX老师。孩子已经适应了，换老师可能要重新磨合。"
> `#损失规避` `#承诺一致` [UID: SL-SEG-D02]

**成交信号**：
- 主动问续费
- 关心下阶段内容
- 推荐朋友来
- 参加机构活动

## 六、客户识别技巧

### 6.1 快速识别方法

**通过提问识别**：

| 问题 | A类小白 | B类焦虑 | C类理性 | D类老客 |
|------|---------|---------|---------|---------|
| "学这个有什么用？" | ✓ | | | |
| "别的孩子都在学吗？" | | ✓ | | |
| "和XX机构有什么区别？" | | | ✓ | |
| "孩子最近进步怎么样？" | | | | ✓ |

**通过行为识别**：

| 行为 | 客户类型 |
|------|----------|
| 频繁看手机、心不在焉 | A类（不感兴趣）或 C类（在对比） |
| 认真记笔记、详细询问 | C类理性家长 |
| 让孩子试玩、拍照 | A类或B类（感性决策） |
| 主动提起之前的课程 | D类老客户 |

### 6.2 沟通切换技巧

**识别错误时的调整**：

| 原判断 | 实际类型 | 调整策略 |
|--------|----------|----------|
| A类→C类 | 发现家长很理性 | 增加数据和逻辑 |
| B类→A类 | 发现家长不焦虑 | 减少紧迫感，增加具象化 |
| C类→B类 | 发现家长有焦虑 | 增加社会认同 |
