---

### 步骤2：任务分析（15分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "遥控器的秘密是'红外通讯'！
>
> 红外通讯的原理：
> 1. 遥控器发出红外光（人眼看不见）
> 2. 红外光携带'编码'信息（比如'开机'='1001'）
> 3. 接收器收到信号，解码后执行命令
>
> 我们用两个设备：
> - **发射端**：按按钮 → 发送红外信号
> - **接收端**：收到信号 → 执行动作
>
> 在Scratch里，我们用'广播'来模拟这个过程：
> - 发射端：广播消息'开灯'
> - 接收端：当收到'开灯'消息时，执行开灯动作"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 信号收不到 | 检查对准 | 红外需要对准，不能有遮挡 |
| 想增加功能 | 多按钮控制 | 不同按钮发送不同命令 |
| 理解快 | 添加反馈 | 接收端收到后闪灯确认 |

**核心代码模式**
```
// 发射端代码
当按下 [按钮1] 时
红外发送 [1]  // 发送编码1

当按下 [按钮2] 时
红外发送 [2]  // 发送编码2

// 接收端代码
当绿旗被点击
重复执行
  如果 <(红外接收) = [1]> 那么
    RGB灯设为 [红色]
  结束
  如果 <(红外接收) = [2]> 那么
    RGB灯设为 [蓝色]
  结束
结束
```

**禁忌提醒**
- 不要让红外发射器和接收器距离太远——信号会衰减
- 不要在强光下测试——阳光会干扰红外信号

---

## 核心知识点

- **红外通讯**：无线数据传输方式
- **编码/解码**：信息的发送和接收
- **广播机制**：一对多的消息传递
