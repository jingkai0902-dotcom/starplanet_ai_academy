## 详细教学流程

### 步骤1：情境引入（5分钟）
`#低负荷-热身` `#IFC-预防`

**教师话术**
> "小朋友们，你们注意过路灯吗？有的路灯整晚都亮着，很浪费电！如果路灯能在有人经过时亮起来，没人时变暗，是不是更节能？今天我们来做智能路灯系统！"

---

### 步骤2：任务分析（15分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "智能路灯的节能逻辑：
> 1. 白天不亮（光线传感器检测）
> 2. 晚上有人经过时全亮（超声波检测）
> 3. 晚上没人时微亮（节能模式）
>
> 亮度分三档：
> - 关闭：白天
> - 微亮（30%）：晚上无人
> - 全亮（100%）：晚上有人
>
> 这就是'智慧城市'的概念——用科技节约能源！"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 检测不准 | 调整距离 | 超声波检测范围设为5米 |
| 想增加功能 | 添加渐变 | 亮度渐变而不是突变 |
| 想增加功能 | 添加统计 | 统计节约了多少电 |

**核心代码模式**
```
当绿旗被点击
重复执行
  如果 <(光线) > (500)> 那么
    // 白天，关灯
    LED亮度设为 (0)
    说 [白天模式：关闭]
  否则
    // 晚上
    如果 <(超声波距离) < (500)> 那么
      // 有人，全亮
      LED亮度设为 (255)
      说 [检测到行人：全亮]
      等待 (10) 秒  // 保持10秒
    否则
      // 无人，微亮
      LED亮度设为 (50)
      说 [节能模式：微亮]
    结束
  结束
  等待 (0.5) 秒
结束
```

- **亮度调节**：PWM控制LED亮度
- **节能设计**：根据需求调整功率
