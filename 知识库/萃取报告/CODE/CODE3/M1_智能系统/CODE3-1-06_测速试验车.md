### 步骤2：任务分析（15分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "测速的原理是物理公式：**速度 = 距离 ÷ 时间**
>
> 我们的测速方案：
> 1. 设置两个红外传感器，相距50厘米
> 2. 小车经过第一个传感器时，开始计时
> 3. 小车经过第二个传感器时，停止计时
> 4. 用公式计算速度
>
> 例如：
> - 距离 = 50厘米 = 0.5米
> - 时间 = 2秒
> - 速度 = 0.5 ÷ 2 = 0.25米/秒 = 25厘米/秒
>
> 还可以换算成'公里/小时'：
> - 0.25米/秒 × 3.6 = 0.9公里/小时"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 计时不准 | 增加传感器间距 | 距离越大，误差影响越小 |
| 想增加功能 | 添加排行榜 | 记录最快速度 |
| 数学困难 | 简化单位 | 只用厘米/秒，不换算 |

**核心代码模式**
```
当绿旗被点击
将 [距离] 设为 (50)  // 两个传感器间距50厘米
将 [计时中] 设为 [否]

// 第一个传感器：开始计时
当 <红外传感器1检测到?> 时
将 [开始时间] 设为 (计时器)
将 [计时中] 设为 [是]

// 第二个传感器：结束计时并计算
当 <红外传感器2检测到?> 时
如果 <(计时中) = [是]> 那么
  将 [结束时间] 设为 (计时器)
  将 [用时] 设为 ((结束时间) - (开始时间))
  将 [速度] 设为 ((距离) / (用时))
  说 (连接 [速度：] 和 (速度) 和 [厘米/秒])
  将 [计时中] 设为 [否]
结束
```

## 核心知识点

- **物理公式**：速度 = 距离 / 时间
- **计时器**：测量时间间隔
- **单位换算**：厘米/秒 → 米/秒 → 公里/小时

**底层逻辑**
- **跨学科融合**：物理+数学+编程
- **实验设计**：控制变量、测量、计算
- **科学方法**：假设→实验→验证
