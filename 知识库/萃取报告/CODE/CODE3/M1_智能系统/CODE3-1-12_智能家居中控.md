---

### 步骤2：任务分析（15分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "智能家居中控系统整合了多个子系统：
>
> 1. **智能照明**（第4课楼梯灯）
>    - 光线暗+有人 → 开灯
>
> 2. **安防系统**（第5课家园卫士）
>    - 雷达扫描 → 发现入侵报警
>
> 3. **环境监控**（第9课智能温室）
>    - 温度、湿度、光线监控
>
> 4. **安全系统**（第11课火灾通道门）
>    - 紧急情况优先处理
>
> 中控系统的作用：
> - 统一管理所有子系统
> - 显示各系统状态
> - 处理优先级冲突"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 系统太复杂 | 简化功能 | 只整合2-3个子系统 |
| 想增加功能 | 添加语音 | 语音播报系统状态 |
| 想增加功能 | 添加遥控 | 用红外遥控器控制 |

**核心代码模式**
```
当绿旗被点击
// 初始化所有子系统
将 [照明状态] 设为 [关]
将 [安防状态] 设为 [正常]
将 [环境状态] 设为 [正常]
将 [安全状态] 设为 [正常]

重复执行
  // 优先级1：安全系统（最高优先级）
  如果 <(温度) > (60)> 那么
    将 [安全状态] 设为 [紧急]
    广播 [紧急疏散]
    // 所有门自动打开
    舵机转到 (90) 度
    蜂鸣器响
    RGB灯设为 [红色]
  否则
    将 [安全状态] 设为 [正常]

    // 优先级2：安防系统
    如果 <(超声波距离) < (50)> 那么
      将 [安防状态] 设为 [警报]
      蜂鸣器响
    否则
      将 [安防状态] 设为 [正常]

      // 优先级3：照明系统
      如果 <<(光线) < (200)> 且 <红外检测到?>> 那么
        将 [照明状态] 设为 [开]
        RGB灯设为 [白色]
      否则
        将 [照明状态] 设为 [关]
        RGB灯设为 [关闭]
      结束

      // 优先级4：环境监控
      如果 <(温度) > (28)> 那么
        电机开启  // 风扇
      否则
        电机关闭
      结束
    结束
  结束

  // 更新显示
  说 (连接 [照明:] 和 (照明状态) 和 [ 安防:] 和 (安防状态))
  等待 (1) 秒
结束
```

## 核心知识点

- **系统集成**：多个子系统协同工作
- **优先级管理**：不同功能有不同优先级
- **状态管理**：用变量记录各系统状态

---

## 单元总结

完成本单元后，学生将掌握：
1. **传感器**：光线、红外、超声波、温度、湿度
2. **执行器**：舵机、电机、RGB灯、蜂鸣器
3. **逻辑设计**：流程图、逻辑运算、优先级
4. **系统思维**：多传感器协同、状态管理
5. **安全设计**：安全逻辑优先权
