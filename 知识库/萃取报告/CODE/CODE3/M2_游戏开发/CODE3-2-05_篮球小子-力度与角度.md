## 详细教学流程

### 步骤1：项目回顾（5分钟）

`#低负荷-热身` `#IFC-预防`

**教师话术**
> "上节课我们做了状态机，运动员可以运球和投篮了。但是投篮的力度和角度怎么控制呢？今天我们来加入力度条和角度控制！"

---

### 步骤2：任务分析（15分钟）

`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "力度控制的原理：
> 1. 屏幕上有一个来回移动的游标
> 2. 游标在左边 → 力度小
> 3. 游标在右边 → 力度大
> 4. 按空格键时，记录游标位置作为力度
>
> 角度控制也类似：
> 1. 另一个游标控制投篮角度
> 2. 游标位置决定球飞出的方向
>
> 这样玩家需要把握时机，在合适的位置按下空格���"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 游标太快 | 降低移动速度 | 从5步改为3步 |
| 想简化操作 | 只控制力度 | 角度固定为45度 |
| 想增加难度 | 游标加速 | 游标速度逐渐变快 |

**核心代码模式**
```
// 力度游标（力度条角色）
当绿旗被点击
移到 x: (-100) y: (-150)
将 [方向] 设为 (1)
重复执行
  如果 <(游戏状态) = [运球]> 那么
    将 x 增加 ((方向) * 5)
    如果 <(x) > (100)> 那么
      将 [方向] 设为 (-1)
    结束
    如果 <(x) < (-100)> 那么
      将 [方向] 设为 (1)
    结束
  结束
结束

// 记录力度
当按下 [空格] 键
如果 <(游戏状态) = [运球]> 那么
  将 [投篮力度] 设为 ((x) + 100)  // 0-200的范围
  将 [投篮力度] 设为 ((投篮力度) / 20)  // 转换为0-10的力度
结束
```

---

### 步骤3：角度控制（15分钟）

`#中负荷-操练` `#IFC-即时`

**教师话术**
> "角度控制的代码和力度类似，但我们用Y坐标来表示角度：
> - Y坐标高 → 角度大（高抛）
> - Y坐标低 → 角度小（平射）
>
> 投篮角度一般在30度到70度之间效果最好！"

**核心代码模式**
```
// 角度游标
当绿旗被点击
移到 x: (-150) y: (0)
将 [角度方向] 设为 (1)
重复执行
  如果 <(游戏状态) = [瞄准角度]> 那么
    将 y 增加 ((角度方向) * 3)
    如果 <(y) > (50)> 那么
      将 [角度方向] 设为 (-1)
    结束
    如果 <(y) < (-50)> 那么
      将 [角度方向] 设为 (1)
    结束
  结束
结束

// 记录角度
当按下 [空格] 键
如果 <(游戏状态) = [瞄准角度]> 那么
  将 [投篮角度] 设为 (30 + ((y) + 50) * 0.4)  // 30-70度
结束
```

## 核心知识点

- 游标系统：来回移动的指示器
- 力度控制：游标位置决定力度大小
- 角度控制：游标位置决定投篮角度
- 时机把握：玩家需要在合适时机按键
