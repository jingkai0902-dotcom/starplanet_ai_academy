## 详细教学流程

### 步骤1：情境引入（5分钟）

**教师话术**
> "小朋友们，想不想开赛车？今天我们用电位器做一个方向盘，控制赛车在公路上飞驰！"

---

### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "电位器是什么？就是一个可以旋转的旋钮！
>
> 旋转电位器，它会输出一个数字（0-1023）：
> - 转到最左边 → 0
> - 转到中间 → 512
> - 转到最右边 → 1023
>
> 我们把这个数字'映射'到赛车的位置：
> - 0 → 赛车在最左边
> - 1023 → 赛车在最右边
>
> 背景滚动的技巧：让背景图片不断向下移动，看起来就像赛车在前进！"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 控制不灵敏 | 调整映射范围 | 缩小有效范围 |
| 想增加难度 | 添加障碍物 | 躲避路上的障碍 |
| 想增加功能 | 添加加速 | 按按钮加速 |

---

### 步骤3：编程设计（25分钟）
`#高负荷-产出` `#IFC-即时`

**核心代码模式**
```
当绿旗被点击
重复执行
  // 电位器控制赛车位置
  将 x 设为 (((电位器值) - 512) / 512 * 200)
  // 电位器512时x=0（中间），0时x=-200（左），1023时x=200（右）
结束

// 背景滚动（背景角色）
当绿旗被点击
重复执行
  将 y 增加 (-5)
  如果 <(y) < (-360)> 那么
    将 y 设为 (360)
  结束
结束
```

---

## 核心知识点

- **电位器**：旋转式模拟量输入（0-1023）
- **数值映射**：将一个范围的值转换到另一个范围
- **背景滚动**：制造运动的视觉效果
