## 详细教学流程

### 步骤1：情境引入（5分钟）
`#低负荷-热身` `#IFC-预防`

**教师话术**
> "夏天好热啊！如果有一个风扇，天热了自动转快，天凉了自动转慢，是不是很棒？今天我们就来做一个智能风扇！"

---

### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "智能风扇需要'感知'温度，所以我们用温度传感器！
>
> 温度传感器会返回当前温度（摄氏度）。
>
> 我们设计三档风速：
> - 温度 < 25°C → 风扇不转
> - 25°C ≤ 温度 < 30°C → 低速
> - 温度 ≥ 30°C → 高速
>
> 这就是'分段函数'的概念——不同范围有不同的处理方式！"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 温度变化慢 | 用手捂传感器 | 体温可以让温度上升 |
| 想增加档位 | 添加中速档 | 四档或五档风速 |
| 想增加功能 | 添加显示 | 显示当前温度和档位 |

---

### 步骤3：编程设计（25分钟）
`#高负荷-产出` `#IFC-即时`

**核心代码模式**
```
当绿旗被点击
重复执行
  如果 <(温度) < (25)> 那么
    电机停止
    说 [风扇关闭]
  否则
    如果 <(温度) < (30)> 那么
      电机功率设为 (50)
      说 [低速运转]
    否则
      电机功率设为 (100)
      说 [高速运转]
    结束
  结束
  等待 (1) 秒
结束
```

---

## 核心知识点

- **温度传感器**：测量环境温度
- **分段控制**：不同温度范围对应不同动作
- **电机调速**：通过功率控制转速
