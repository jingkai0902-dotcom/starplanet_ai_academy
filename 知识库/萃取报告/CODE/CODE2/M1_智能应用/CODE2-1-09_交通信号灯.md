---

### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "交通信号灯有三种颜色：红、黄、绿。我们用RGB灯来实现！
>
> RGB是什么？R=红(Red)，G=绿(Green)，B=蓝(Blue)。
> 每种颜色的亮度从0到255：
> - 红灯：R=255, G=0, B=0
> - 绿灯：R=0, G=255, B=0
> - 黄灯：R=255, G=255, B=0（红+绿=黄）
>
> 倒计时用四位数码管显示。
>
> 信号灯的逻辑：
> 1. 绿灯亮30秒
> 2. 黄灯亮3秒（警告）
> 3. 红灯亮30秒
> 4. 循环"

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| RGB混淆 | 用调色板演示 | 展示颜色混合效果 |
| 想增加功能 | 添加行人灯 | 红灯时行人绿灯 |
| 想增加难度 | 添加紧急模式 | 按钮触发全红灯 |

---

### 步骤3：编程设计（25分钟）
`#高负荷-产出` `#IFC-即时`

**核心代码模式**
```
当绿旗被点击
重复执行
  // 绿灯30秒
  RGB灯设为 (0) (255) (0)
  重复 (30) 次
    数码管显示 (30 - 计数器)
    等待 (1) 秒
  结束

  // 黄灯3秒
  RGB灯设为 (255) (255) (0)
  重复 (3) 次
    数码管显示 (3 - 计数器)
    等待 (1) 秒
  结束

  // 红灯30秒
  RGB灯设为 (255) (0) (0)
  重复 (30) 次
    数码管显示 (30 - 计数器)
    等待 (1) 秒
  结束
结束
```

---

## 核心知识点

- **RGB颜色模型**：三原色混合（0-255）
- **四位数码管**：显示数字
- **状态循环**：绿→黄→红→绿...
