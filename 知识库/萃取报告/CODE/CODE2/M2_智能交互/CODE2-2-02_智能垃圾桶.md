## 详细教学流程

**步骤1：情境引入（5分钟）** `#低负荷-热身` `#IFC-预防`

**教师话术**
> "小朋友们，你们扔垃圾的时候是不是要用手掀开盖子？手脏了怎么办？今天我们来做一个智能垃圾桶，手一靠近盖子就自动打开！"

---

**步骤2：任务分析（15分钟）** `#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "智能垃圾桶需要两个部件：
> 1. **红外传感器**：检测有没有手靠近
> 2. **舵机**：控制盖子开合
>
> 工作流程：
> - 红外检测到手 → 舵机转到90度（盖子打开）
> - 红外没检测到 → 等待2秒 → 舵机转到0度（盖子关闭）
>
> 为什么要等2秒？给你时间把垃圾扔进去啊！"

**核心代码模式**
```
当绿旗被点击
舵机转到 (0) 度  // 初始关闭
重复执行
  如果 <红外传感器检测到?> 那么
    舵机转到 (90) 度  // 打开
    等待直到 <红外传感器未检测到?>
    等待 (2) 秒
    舵机转到 (0) 度   // 关闭
  结束
结束
```

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 盖子开合太快 | 添加缓动 | 分多步转动，每步等待0.1秒 |
| 想增加功能 | 添加语音 | 打开时说"谢谢" |
| 红外不灵敏 | 调整位置 | 确保传感器朝向正确 |

- 舵机控制：精确角度定位
