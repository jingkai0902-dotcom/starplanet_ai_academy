## 详细教学流程

**步骤1：课堂问候（2分钟）** `#低负荷-热身` `#IFC-预防`

**教师话术**
> "小朋友们好！你们见过机场的行李跟随车吗？它能自动跟着主人走，不会撞上去也不会掉队。今天我们就来做一个自动跟随小车！"

---

**步骤2：情境引入（5分钟）** `#低负荷-热身` `#IFC-预防`

**教师话术**
> "【播放动画】斯斯去超市买了很多东西，提着太重了。他想：要是有一个小车能自动跟着我就好了！不用拉不用推，它自己就会跟上来。"

---

**步骤3：任务分析（15分钟）** `#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "自动跟随小车的逻辑比较复杂，我们来分析：
>
> 1. **太远了**（距离 > 30cm）：加速追上去
> 2. **刚刚好**（15cm < 距离 < 30cm）：保持速度跟随
> 3. **太近了**（距离 < 15cm）：减速或停止
>
> 这就是三种状态的判断！我们用超声波传感器测距离，用直流电机控制速度。"

**核心代码模式**
```
当绿旗被点击
重复执行
  如果 <(超声波距离) > (30)> 那么
    电机速度设为 (100)  // 全速追
  否则
    如果 <<(超声波距离) > (15)> 且 <(超声波距离) <= (30)>> 那么
      电机速度设为 (60)  // 中速跟随
    否则
      如果 <(超声波距离) <= (15)> 那么
        电机速度设为 (0)  // 停止
      结束
    结束
  结束
结束
```

**变体示例**

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 小车反应慢 | 减少等待时间 | 提高检测频率 |
| 跟随不稳定 | 调整阈值 | 根据实际测试微调距离 |
| 想增加功能 | 添加转向 | 用两个超声波实现左右跟随 |

**禁忌提醒**
- 不要让小车速度太快——容易撞到障碍物
- 不要在反光地面测试——超声波可能误判
- 不要忘记给电机供电——电机需要独立电源

## 核心知识点

- 超声波传感器：精确测距（2-400cm）
- 直流电机：速度控制（0-100%）
- 多重判断：if-else-if结构
- 控制闭环：感知→判断→执行→反馈

**底层逻辑**
- **反馈控制**：根据距离实时调整速度
- **状态机**：三种状态（追赶/跟随/停止）
- **PID思想初探**：距离越远速度越快（比例控制）
