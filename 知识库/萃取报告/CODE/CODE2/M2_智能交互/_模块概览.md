## 单元概览

| 项目 | 内容 |
|------|------|
| **单元编号** | 2-2 |
| **单元名称** | 智能交互系统 (Smart Interactive Systems) |
| **适用年龄** | 二年级（7-8岁） |
| **课时数** | 12节 × 90分钟 |
| **核心目标** | 掌握各类传感器与执行器的协同工作，理解控制闭环 |
| **主要教具** | 触碰开关、红外传感器、超声波传感器、声音传感器、舵机、蜂鸣器、RGB灯 |

**底层逻辑**
CODE2-2是该阶段的核心模块，重点让学生理解"输入（感知）- 处理（逻辑）- 输出（执行）"的计算机控制闭环。

**单元教学策略**
- 基础控制（01-04）：触碰、红外、数码管
- 进阶应用（05-08）：蜂鸣器、超声波、逻辑运算
- 综合系统（09-12）：语音合成、声音传感器、舵机

## 教学禁忌清单

| 序号 | 禁忌 | 原因 | 正确做法 |
|------|------|------|----------|
| 1 | 直接告诉答案 | 剥夺学生思考机会 | 通过提问引导学生思考 |
| 2 | "看一步做一步" | 学生不理解逻辑 | 先理解逻辑流程图再动手 |
| 3 | 家长会死板 | 错失建立信任机会 | 具体表扬（如"XX同学搭建结构很稳固"） |
| 4 | 只关注软件或只关注硬件 | 软硬结合才是重点 | 既要搭好模型，也要调试硬件参数 |
| 5 | 强行扭动舵机 | 损坏内部齿轮 | 不要手动强行转动舵机 |

## 底层教育学原理

| 原理 | 说明 | 在本单元的应用 |
|------|------|----------------|
| **控制闭环** | 输入-处理-输出的完整循环 | 传感器检测→程序判断→执行器动作 |
| **软硬结合** | 编程与物理实体结合 | 程序控制真实的舵机、电机、传感器 |
| **循环嵌套** | 高级编程逻辑 | SOS信号用嵌套循环简化代码 |
| **逻辑运算** | 布尔代数基础 | 逻辑与、逻辑非的实际应用 |
