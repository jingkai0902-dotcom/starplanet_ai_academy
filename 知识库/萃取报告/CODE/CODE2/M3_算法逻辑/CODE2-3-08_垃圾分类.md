## 详细教学流程

### 步骤1：情境引入（5分钟）
`#低负荷-热身` `#IFC-预防`

**教师话术**
> "【播放动画】斯斯的小区开始实行垃圾分类了！可回收垃圾要放蓝色桶，厨余垃圾放绿色桶。可是有些垃圾不知道该放哪个桶，怎么办呢？"

---

### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "什么是'或'呢？用口语表达就是'或者'。
>
> 例如：如果是塑料瓶**或者**是纸张**或者**是金属，就放进可回收桶。
> 只要满足其中一个条件就行！
>
> 和'与'的区别：
> - '与'：所有条件都要满足
> - '或'：只要一个条件满足就行"

---

## 核心代码模式

```
如果 <<<(垃圾类型) = [塑料瓶]> 或 <(垃圾类型) = [纸张]>> 或 <(垃圾类型) = [金属]>> 那么
  说 [请放入可回收垃圾桶（蓝色）]
否则
  如果 <<(垃圾类型) = [剩饭]> 或 <(垃圾类型) = [果皮]>> 那么
    说 [请放入厨余垃圾桶（绿色）]
  结束
结束
```

## 变体示例

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 混淆与/或 | 画真值表 | 对比AND和OR的结果 |
| 条件太多 | 分组处理 | 先判断大类再判断小类 |
| 理解快 | 添加有害垃圾 | 增加第三种分类 |

---

## 核心知识点

- **布尔运算-或(OR)**：条件A成立**或**条件B成立，结果就为真
- **与OR的区别**：AND要全部满足，OR只需一个满足
