### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "什么是'非'呢？就是'取反'、'相反'的意思。
>
> 例如：静音按钮
> - 当前是响的 → 按一下变成静音
> - 当前是静音 → 按一下变成响的
>
> 用'非'运算：`静音 = 非(静音)`
> 如果静音是真，非(真)就是假
> 如果静音是假，非(假)就是真"

---

## 核心代码模式

```
当按下 [m] 键
将 [静音] 设为 <非 (静音)>

// 播放音符时检查
如果 <非 (静音)> 那么
  播放音符 (60)
结束
```

## 核心知识点

- **布尔运算-非(NOT)**：取反，真变假，假变真
- **状态切换**：用NOT实现开关效果
