## 详细教学流程

### 步骤1：情境引入（5分钟）
`#低负荷-热身` `#IFC-预防`

**教师话术**
> "【播放动画】斯斯来到了海边，看到渔民在捕鱼。不同的鱼价值不同：小鱼1分，中鱼2分，大鱼5分。在限定时间内，看谁能得到最高分！"

---

### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "捕鱼游戏的计分逻辑：
> - 捕到小鱼：得分 + 1×1 = 1分
> - 捕到中鱼：得分 + 1×2 = 2分
> - 捕到大鱼：得分 + 1×5 = 5分
>
> 这里用到了乘法运算！每种鱼有自己的'分值系数'。"

---

## 核心代码模式

```
// 鱼的脚本
当作为克隆体启动时
移到随机位置
显示
重复执行
  移动 (2) 步
  碰到边缘就反弹
  如果 <碰到 [网] ?> 那么
    将 [得分] 增加 (分值)  // 分值是这条鱼的系数
    删除此克隆体
  结束
结束
```

## 变体示例

| 学生情况 | 调整方案 | 说明 |
|----------|----------|------|
| 鱼太少 | 增加克隆数量 | 用循环创建更多鱼 |
| 想增加难度 | 添加炸弹鱼 | 碰到炸弹鱼扣分 |
| 理解快 | 添加连击奖励 | 连续捕鱼有额外加成 |

## 核心知识点

- **乘法运算**：`得分 = 数量 × 分值系数`
- **计时器**：限时游戏机制
- **克隆技术**：批量创建相似对象
