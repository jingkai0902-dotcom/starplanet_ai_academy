## 详细教学流程

### 步骤1：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "上节课我们做了基础计算器，今天要升级！
>
> 新功能：
> 1. **连续计算**：1+2=3，然后+4=7，不用重新输入3
> 2. **清除功能**：C键清除当前输入，AC键全部清除
> 3. **错误处理**：除以0时显示'Error'"

---

## 核心代码模式

```
// 连续计算：把结果作为下一次的第一个数
当按下 [=] 键
计算结果
将 [数字1] 设为 (结果)
将 [数字2] 设为 []
将 [输入状态] 设为 [等待运算符]

// 除以0的错误处理
如果 <<(运算符) = [÷]> 且 <(数字2) = [0]>> 那么
  显示 [Error]
  停止
结束
```
