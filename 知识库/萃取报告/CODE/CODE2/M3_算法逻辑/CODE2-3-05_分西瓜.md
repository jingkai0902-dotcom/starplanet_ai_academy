## 详细教学流程

### 步骤1：情境引入（5分钟）
`#低负荷-热身` `#IFC-预防`

**教师话术**
> "【播放动画】斯斯买了一堆西瓜要分给小伙伴们。10个西瓜分给3个人，每人能分几个？还剩几个？"

---

### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "除法运算有两个结果：
> 1. **商**：每人分到几个（10÷3=3...1，商是3）
> 2. **余数**：剩下几个（余数是1）
>
> 在Scratch里：
> - `10 / 3` 得到 3.33...（带小数）
> - `10 mod 3` 得到 1（余数）
> - 要取整数部分，用 `取整(10/3)` 得到 3"

---

## 核心代码模式

```
当绿旗被点击
询问 [有多少个西瓜？] 并等待
将 [西瓜数] 设为 (回答)
询问 [要分给几个人？] 并等待
将 [人数] 设为 (回答)
将 [每人分到] 设为 (取整 ((西瓜数) / (人数)))
将 [剩余] 设为 ((西瓜数) mod (人数))
说 (连接 [每人分到] 和 (每人分到) 和 [个，剩余] 和 (剩余) 和 [个])
```

## 核心知识点

- **除法运算**：`商 = 被除数 / 除数`
- **取整运算**：去掉小数部分
- **取余运算(mod)**：得到余数
