---

### 步骤2：任务分析（20分钟）
`#中负荷-操练` `#IFC-即时` ⭐核心环节

**教师话术**
> "计算器的核心逻辑：
> 1. **输入数字**：按0-9键，数字显示在屏幕上
> 2. **选择运算**：按+、-、×、÷键
> 3. **输入第二个数**：继续按数字键
> 4. **得出结果**：按=键计算
>
> 难点是：怎么把多次按键组合成一个多位数？
> 比如按1、2、3，要变成123，不是1+2+3=6！"

---

## 核心代码模式

```
// 输入数字的逻辑
当按下 [1] 键
如果 <(输入状态) = [第一个数]> 那么
  将 [数字1] 设为 (连接 (数字1) 和 [1])
否则
  将 [数字2] 设为 (连接 (数字2) 和 [1])
结束
更新显示

// 按等号计算
当按下 [=] 键
如果 <(运算符) = [+]> 那么
  将 [结果] 设为 ((数字1) + (数字2))
结束
显示 (结果)
```

## 核心知识点

- **字符串拼接**：把多次输入组合成多位数
- **状态控制**：区分"输入第一个数"和"输入第二个数"
- **运算符存储**：记住用户选择的运算类型
