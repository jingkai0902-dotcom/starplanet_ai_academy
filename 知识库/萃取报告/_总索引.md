## 四、目录结构优化

### 拆分前（模块级）
```
萃取报告/
├── CODE/CODE1_01_机械结构.md  # 1000行，12节课
├── CODE/CODE1_02_智能硬件.md  # 750行，12节课
└── ...
```

### 拆分后（单节课级）
```
萃取报告/
├── CODE/CODE1/M1_机械结构/
│   ├── _模块概览.md           # 模块导航
│   ├── CODE1-1-01_旋转的小鸟.md  # 单节课
│   ├── CODE1-1-02_高低传送带.md  # 单节课
│   └── ...
└── ...
```

### 优化效果
- **精准查找**：直接定位到具体课程文件
- **独立维护**：修改单节课不影响其他课程
- **AI检索友好**：文件名包含课程编号和名称
