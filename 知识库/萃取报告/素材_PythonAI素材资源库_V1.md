## 一、Python AI课程体系概览

### 1.1 课程阶段总览

| 阶段 | 名称 | 适用年龄 | 年级 | 核心方向 |

| Python AI 1 | 人工智能基础 | 9-11岁 | 4-6年级 | Python语法入门、智能硬件启蒙 |
| Python AI 2 | 人工智能进阶 | 10-12岁 | 5-7年级 | 数据结构、计算机视觉基础 |

## 五、教学理念与方法论

### 5.1 PBL项目制学习

**核心理念**："为理解而教，为创新而学"

**认知发展模型（点-线-面-体）**：

| 层级 | 名称 | 培养能力 |
|------|------|----------|
| 点 | 数据 | 分解能力 |
| 线 | 逻辑/算法 | 模式识别与算法设计能力 |
| 面 | 结构/抽象 | 抽象与模块化思维 |
| 体 | 系统 | 系统思维与全局观 |

### 5.2 八大能力与5E教学法

**八大能力**：
通过Roleplay测评收集现象，解读孩子的：
1. 观察力
2. 分析力
3. 理解力
4. 想象力
5. 判断力
6. 推理力
7. 动手能力
8. 语言能力

**5E教学法**：
1. 参与（Engage）
2. 探索（Explore）
3. 解释（Explain）
4. 拓展（Elaborate）
5. 评价（Evaluate）

## 六、师资培训与家长沟通体系

### 6.1 教师内训

- 教师需成为"教育理念的传递者"
- 不仅教代码，更要传达"拨开人群，解决问题"的价值观
- 通过"点线面体"理论向家长解释编程对思维的提升
- 通过"滞后性"理论解释教育效果的长期性

### 6.2 咨询话术逻辑

**公式**：
```
现象（如写作业磨蹭）→ 做法（键盘操作训练）→ 能力（手眼协调）→ 好处（卷面分提高）
```

**差异化沟通策略**：
| 家长类型 | 沟通重点 |
|----------|----------|
| 小白家长 | 通俗易懂，强调实际效果 |
| 高知家长 | 专业术语，强调底层逻辑 |
| 老客户 | 强调进阶价值，续费转化 |
