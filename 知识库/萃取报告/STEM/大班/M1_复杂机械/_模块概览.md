# STEM大班 M1 复杂机械 模块概览

> **适用年龄**：5-6岁大班
> **课程数量**：12节（STEM-3-1-01 至 STEM-3-1-12）
> **核心主题**：基础机械原理（杠杆、齿轮、轮轴、滑轮等）
> **教学目标**：通过搭建和实验，让幼儿理解简单机械的工作原理

---

## 课程列表

| 编号 | 课程名称 | 核心技能 | 认知负荷 |
|------|---------|---------|---------|
| STEM-3-1-01 | 杠杆与平衡 | 杠杆原理、支点、力臂、平衡 | 中等 |
| STEM-3-1-02 | 齿轮传动 | 齿轮啮合、转向变化、速度变化 | 中等 |
| STEM-3-1-03 | 轮轴原理 | 轮和轴的关系、省力原理 | 中等 |
| STEM-3-1-04 | 滑轮系统 | 定滑轮、动滑轮、滑轮组 | 中高 |
| STEM-3-1-05 | 斜面与螺旋 | 斜面省力原理、螺旋结构 | 中等 |
| STEM-3-1-06 | 摩擦力探究 | 摩擦力产生、影响因素 | 中等 |
| STEM-3-1-07 | 弹性与弹力 | 弹性材料、弹力大小 | 中等 |
| STEM-3-1-08 | 重心与稳定 | 重心位置、底面积与稳定性 | 中等 |
| STEM-3-1-09 | 连杆机构 | 连杆传动、运动轨迹变换 | 中高 |
| STEM-3-1-10 | 凸轮机构 | 凸轮形状与运动规律 | 中高 |
| STEM-3-1-11 | 棘轮与棘爪 | 单向传动、防逆转机构 | 中高 |
| STEM-3-1-12 | 综合机械挑战 | 多种简单机械组合应用 | 高 |
