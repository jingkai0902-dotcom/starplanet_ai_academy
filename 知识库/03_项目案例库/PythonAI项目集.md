## 精选项目详解

### 贪食蛇（PYAI 2-2-8）

**项目简介**：经典贪食蛇游戏，控制蛇吃食物，身体变长。

**核心技术**：
- 列表：存储蛇身坐标
- 碰撞检测：吃食物、撞墙
- 键盘控制：方向控制
- 游戏循环：Pygame主循环

**核心代码**：
```python
import pygame
import random

class Snake:
    def __init__(self):
        self.body = [(100, 100), (80, 100), (60, 100)]
        self.direction = (20, 0)

    def move(self):
        head = self.body[0]
        new_head = (head[0] + self.direction[0],
                    head[1] + self.direction[1])
        self.body.insert(0, new_head)
        self.body.pop()  # 移除尾部

    def grow(self):
        # 吃到食物时不移除尾部
        head = self.body[0]
        new_head = (head[0] + self.direction[0],
                    head[1] + self.direction[1])
        self.body.insert(0, new_head)
```

### 手势控制游戏（PYAI 2-3-8）

**项目简介**：用手势控制游戏角色移动。

**核心技术**：
- MediaPipe Hands：手部检测
- 手势映射：手势→指令
- Pygame：游戏界面

**核心代码**：
```python
import cv2
import mediapipe as mp

mp_hands = mp.solutions.hands
hands = mp_hands.Hands()

# 手势映射
GESTURE_MAP = {
    (0, 1, 0, 0, 0): 'UP',      # 食指
    (0, 1, 1, 0, 0): 'DOWN',    # 食指+中指
    (1, 1, 1, 1, 1): 'STOP',    # 全部伸开
}

def get_fingers_up(landmarks):
    fingers = []
    # 拇指
    if landmarks[4].x < landmarks[3].x:
        fingers.append(1)
    else:
        fingers.append(0)
    # 其他四指
    for tip in [8, 12, 16, 20]:
        if landmarks[tip].y < landmarks[tip-2].y:
            fingers.append(1)
        else:
            fingers.append(0)
    return tuple(fingers)
```

### 海龟雷达（MP1-L10~12）

**项目简介**：超声波雷达扫描系统，舵机带动超声波传感器旋转扫描，数据通过串口发送到电脑可视化。

**核心技术**：
- HC-SR04：超声波测距
- SG90舵机：角度控制
- UART：串口通信
- turtle：数据可视化

**ESP32端代码**：
```python
from novaMP import HCSR04, SG90
import time

sonar = HCSR04(trig=5, echo=18)
servo = SG90(15)

for angle in range(0, 181, 5):
    servo.angle(angle)
    time.sleep(0.1)
    distance = sonar.distance_cm()
    print(f"{angle},{distance}")  # 发送到串口
```

**电脑端可视化**：
```python
import turtle
import serial

ser = serial.Serial('COM3', 115200)
t = turtle.Turtle()

while True:
    data = ser.readline().decode().strip()
    angle, distance = map(float, data.split(','))
    # 绘制雷达扫描线
    t.setheading(angle)
    t.forward(distance)
    t.backward(distance)
```
