## 精选项目详解

### 荷兰风车转转转（STEM 2-2-10）

**项目简介**：搭建荷兰风车模型，理解风能利用和齿轮传动。

**核心概念**：
- 风能：风吹动叶片旋转
- 齿轮传动：将旋转传递到磨盘
- 能量转化：风能→机械能

**搭建要点**：
1. 底座稳固
2. 叶片角度合适
3. 齿轮啮合正确

**5E教学流程**：
- Engage：荷兰风车图片导入
- Explore：搭建风车模型
- Explain：讲解风能原理
- Elaborate：改变叶片数量观察
- Evaluate：展示作品

---

### 智能人行道（STEM 2-3-10）

**项目简介**：搭建智能人行道模型，行人经过时自动亮灯。

**核心概念**：
- 循环结构：重复检测
- 条件判断：检测到人→亮灯
- 传感器应用：距离传感器

**编程逻辑**：
```
重复执行
    如果 <距离传感器 < 30cm> 那么
        LED 亮
        等待 3 秒
    否则
        LED 灭
```

---

### 雷达扫描仪（STEM 3-4-07）

**项目简介**：舵机带动超声波传感器旋转扫描，检测周围障碍物。

**核心概念**：
- 舵机控制：角度旋转
- 超声波测距：距离检测
- 数据可视化：显示扫描结果

**编程逻辑**：
```
重复执行
    舵机转到 0度
    测量距离
    舵机转到 45度
    测量距离
    舵机转到 90度
    测量距离
    ...
```
