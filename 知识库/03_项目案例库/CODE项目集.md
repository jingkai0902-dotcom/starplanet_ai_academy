## 精选项目详解

### 征服地鼠族（CODE1-3-10）

**项目简介**：经典打地鼠游戏，地鼠随机出现，玩家点击得分。

**核心技术**：
- 变量：记录得分
- 计时器：限时游戏
- 随机数：地鼠随机位置
- 条件判断：点击检测

**核心积木**：
```
当绿旗被点击
设定 [得分] 为 (0)
计时器归零
重复执行直到 <(计时器) > (30)>
    移到 (随机位置)
    等待 (随机数 0.5 到 2) 秒
说 (字符串连接 [得分:] 和 (得分))
```

### 植物大战僵尸（CODE3-2-07~09）

**项目简介**：经典塔防游戏，种植植物防御僵尸入侵。

**核心技术**：
- 列表：存储僵尸数据
- 网格对齐：植物定位
- 克隆技术：批量创建对象
- 射线检测：攻击判定
- 波次系统：难度递增

**核心算法**：
```
// 网格对齐
格子X = 取整((鼠标x + 200) / 80)
格子Y = 取整((鼠标y + 150) / 80)

// 射线检测
如果 <(僵尸行号) = (植物行号)> 那么
    发射子弹
```
