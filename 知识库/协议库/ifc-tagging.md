# 互动反馈循环（IFC）打标协议 V1.0

> 协议编号：7.15
> 创建日期：2026-01-24
> 适用场景：萃取课堂管理技巧时

---

## 一、核心原则

**AI在提取课堂管理技巧时，必须标注该技巧属于"预防性"、"即时性"还是"复盘性"，帮助新教师快速检索到最适合当前状态的应对策略。**

---

## 二、定义与价值

| 概念 | 定义 | 价值 |
|------|------|------|
| **IFC打标** | 按互动反馈循环阶段对技巧进行分类 | 支持场景化检索 |
| **预防性技巧** | 在问题发生前采取的预防措施 | 减少问题发生 |
| **即时性技巧** | 问题发生时的即时应对策略 | 快速解决问题 |
| **复盘性技巧** | 问题解决后的总结和改进 | 防止问题复发 |

---

## 三、IFC标签体系

| 标签 | 定义 | 适用场景 | 示例技巧 |
|------|------|---------|---------|
| `#IFC-预防` | 课前/课中预防性措施 | 备课阶段、课堂开始 | 座位安排、规则建立、活动设计 |
| `#IFC-即时` | 问题发生时的即时应对 | 课堂进行中 | 眼神提醒、暂停技巧、转移注意力 |
| `#IFC-复盘` | 课后总结和长期改进 | 课后、周期性 | 行为记录、家长沟通、策略调整 |

---

## 四、IFC打标使用规范

**课堂管理技巧萃取时，必须按以下格式标注：**

```markdown
### 技巧：[技巧名称]

**IFC分类**：`#IFC-预防` / `#IFC-即时` / `#IFC-复盘`

**适用场景**
- 问题类型：[具体问题]
- 适用年龄：[年龄段]
- 适用时机：[课前/课中/课后]

**操作步骤**
1. [步骤1]
2. [步骤2]
3. [步骤3]

**配套话术**
> "[话术内容]"

**注意事项**
- [注意事项1]
- [注意事项2]

`#执行层` `#IFC-[类型]`
[Source ID: 课堂管理-[技巧ID]]
```

## 五、IFC打标示例

### 示例1：预防性技巧

```markdown
### 技巧：座位安排策略

**IFC分类**：`#IFC-预防`

**适用场景**
- 问题类型：学生注意力不集中、互相干扰
- 适用年龄：K1-G3
- 适用时机：课前准备

**操作步骤**
1. 识别"高风险"学生（注意力差、爱说话）
2. 将高风险学生安排在前排或靠近老师的位置
3. 将安静的学生与活跃的学生交叉安排
4. 避免将好朋友安排在相邻位置
5. 每2-4周调整一次座位

**配套话术**
> "今天我们换一下座位，XX坐到这里来，离老师近一点，老师可以更好地帮助你。"

**注意事项**
- 调整座位时要给出正面理由，避免让学生感到被惩罚
- 观察调整后的效果，必要时再次调整

`#执行层` `#IFC-预防`
[Source ID: 课堂管理-座位安排-01]
```

### 示例2：即时性技巧

```markdown
### 技巧：眼神提醒法

**IFC分类**：`#IFC-即时`

**适用场景**
- 问题类型：学生轻微走神、小动作
- 适用年龄：K2-G3
- 适用时机：课中，问题初现时

**操作步骤**
1. 不中断教学，继续讲课
2. 用眼神锁定走神的学生
3. 保持眼神接触3-5秒
4. 如果学生注意到，微微点头示意
5. 如果学生没注意到，走近该学生继续教学

**配套话术**
（无需话术，纯眼神交流）

**注意事项**
- 眼神要温和但坚定，不要瞪视
- 如果眼神提醒无效，升级到其他即时技巧

`#执行层` `#IFC-即时`
[Source ID: 课堂管理-眼神提醒-01]
```

### 示例3：复盘性技巧

```markdown
### 技巧：行为记录与家长沟通

**IFC分类**：`#IFC-复盘`

**适用场景**
- 问题类型：反复出现的行为问题
- 适用年龄：K1-G3
- 适用时机：课后

**操作步骤**
1. 课后立即记录问题行为（时间、情境、表现）
2. 分析行为模式（是否有规律）
3. 制定改进策略
4. 与家长沟通，寻求配合
5. 下次课观察改进效果

**配套话术**
> "XX妈妈，今天课上XX在XX环节有点坐不住，我观察到他可能是因为XX。我们可以一起试试XX方法，您在家也可以配合一下。"

**注意事项**
- 记录要客观，避免主观评价
- 与家长沟通时先肯定孩子的优点

`#执行层` `#IFC-复盘`
[Source ID: 课堂管理-行为记录-01]
```

## 六、检查清单

**AI在萃取课堂管理技巧时，必须检查：**

- [ ] 每个技巧是否标注了IFC分类？
- [ ] IFC分类是否准确（预防/即时/复盘）？
- [ ] 是否标注了适用时机？
- [ ] 是否提供了配套话术（如适用）？
- [ ] 是否标注了 `#执行层` 标签？
