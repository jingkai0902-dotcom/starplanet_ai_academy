# 考核题关联话术索引协议 V1.0

> 协议编号：7.19
> 创建日期：2026-01-24
> 适用场景：生成销售考核题时

---

## 一、核心原则

**AI为每道销售考核题标注"关联话术索引"，实现"考、评、练"一体化。如果销售人员答错，程序可自动弹出对应的标准话术和逻辑说明。**

---

## 二、定义与价值

| 概念 | 定义 | 价值 |
|------|------|------|
| **话术索引** | 将考核题与话术库中的具体话术关联 | 支持精准补课 |
| **考评练一体化** | 考核→评估→练习的闭环 | 提升培训效果 |
| **自动推送** | 答错后自动推送学习材料 | 提升学习效率 |

---

**每道销售考核题必须包含以下索引信息：**

**标准话术（答错时推送）**

**底层逻辑**

[Source ID: 题库-销售-[题号]]

## 四、关联话术索引示例

```markdown
### 题目：家长说"你们比XX机构贵"，以下哪种回应最合适？

**选项**
- A. "我们的老师都是名校毕业的，所以贵一点。"
- B. "您说得对，我们确实不是最便宜的。但您有没有想过，为什么很多家长在对比了多家机构后，最终还是选择了我们？"
- C. "那您可以去XX机构看看，我们不强求。"
- D. "我们可以给您打个折。"

**答案**：B

**解析**
B选项采用了"先认同再反问"的策略，将价格比较转化为价值比较，引导家长思考选择标准。A选项只强调成本，没有建立价值；C选项放弃客户；D选项直接降价，损害利润且没有建立价值认知。

---

**关联话术索引**

| 索引项 | 内容 |
|--------|------|
| **关联话术** | 价格异议处理话术 |
| **话术位置** | 管理层专区/1-工具模板库/销售话术库.md#价格异议 |
| **话术类型** | 异议处理 |
| **心理学标签** | `#缓解焦虑` `#价值锚定` `#社会认同` |

**标准话术（答错时推送）**
> "您说得对，我们确实不是最便宜的。但您有没有想过，为什么很多家长在对比了多家机构后，最终还是选择了我们？
>
> 其实很多家长一开始也有这个顾虑。但他们后来发现，孩子在励步学一年，词汇量能达到200个，而公立学校一年只教50个。平均下来，每个单词的学习成本其实比补习班还低。
>
> 而且，这些单词孩子是真正掌握了，不是死记硬背的。您觉得，是便宜但学不会划算，还是稍微贵一点但真正学会划算？"

**底层逻辑**
1. **先认同**（`#缓解焦虑`）：降低家长防御心理
2. **再反问**（`#社会认同`）：利用从众心理，暗示"很多家长选择我们"
3. **重构价值**（`#价值锚定`）：用词汇量对比建立价值参照
4. **引导选择**：用二选一问题引导家长思考

**练习建议**
- 建议练习次数：5次以上
- 练习方式：与同事角色扮演，录音后自评
- 通关标准：能流畅说出完整话术，语气自然，能根据家长反应灵活调整

**相关题目**
- 题库-销售-023：家长说"再考虑考虑"
- 题库-销售-025：家长说"线上课更便宜"

`#测评项` `#关联索引`
[Source ID: 题库-销售-018]
```

## 五、检查清单

**AI在生成销售考核题时，必须检查：**

- [ ] 每道题是否标注了关联话术？
- [ ] 话术位置是否准确（含锚点）？
- [ ] 是否提供了完整的标准话术？
- [ ] 是否解释了话术的底层逻辑？
- [ ] 是否标注了心理学标签？
- [ ] 是否提供了练习建议？
- [ ] 是否关联了相关题目？
