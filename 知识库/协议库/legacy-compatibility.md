# 体系兼容性处理协议 V1.0

> 协议编号：7.24
> 创建日期：2026-01-24
> 适用场景：处理新旧课程体系内容时

---

## 一、核心原则

**明确新旧课程体系的关系，在萃取过程中正确处理旧体系内容，保留通用价值，剔除逻辑冲突。**

---

## 二、课程体系定义

### 当前有效课程体系（必须使用）

**三大招生入口**：
| 入口 | 年龄 | 定位 |
|------|------|------|
| **K1** | 3-4岁 | 学龄前零基础入口 |
| **K2** | 4-5岁 | 学龄前零基础入口 |
| **G0** | 5-8岁 | 小学龄零基础入口（20周后进入G1） |

**课程进阶路径**：
```
K1入口：K1 → K2 → K3 ─┐
                      ├→ G1 → G2 → G3 → SS(1-4)
K2入口：K2 → K3 ──────┤
                      │
G0入口：G0(20周) ─────┘
```

| 阶段 | 级别 | 目标年龄 | 学习周期 | 核心定位 |
|------|------|---------|---------|---------|
| **K阶段** | K1, K2, K3 | 3-6岁 | 各1年 | 启蒙阶段，建立语感和兴趣 |
| **G0** | G0 | 5-8岁 | **20周（半年）** | 零基础入口，快速衔接G1 |
| **G阶段** | G1, G2, G3 | 6-10岁 | 各1年 | 小学阶段，系统学习 |
| **SS阶段** | SS1, SS2, SS3, SS4 | 10-13岁 | 各1年 | 高阶阶段，学术英语 |

**G0级别特别说明**：
- **定位**：针对没有励步K系列基础的小学龄儿童（5-8岁零基础）
- **学习周期**：20周（半年），快速衔接G1
- **课程模式**：1节线上AI课 + 2节线下课
- **战略意义**：G0是"零基础入口"，完成后直接进入G1，与K系列学员汇合

### 已淘汰的旧体系（禁止作为课程逻辑引用）

| 旧体系 | 说明 | 处理方式 |
|--------|------|---------|
| **Hip Hip Hooray (3H)** | 旧版教材系列 | 禁止引用教材内容和级别逻辑 |
| **Stage 1-4** | 旧版课程分级 | 禁止引用级别划分逻辑 |
| **Stage 1-10** | 旧版完整课程体系 | 禁止引用级别划分逻辑 |

## 三、旧体系内容处理原则

### 可以保留的内容（通用价值）

| 类型 | 说明 | 处理方式 |
|------|------|---------|
| **教学技术** | PPP模型、M-P-F模型、Story五步法等 | ✅ 保留，这些是通用教学方法论 |
| **课堂管理技巧** | 纪律管理、奖励机制、互动设计等 | ✅ 保留，适用于任何课程体系 |
| **话术模板** | 家长沟通、异议处理、销售话术等 | ✅ 保留，但需去除旧体系级别引用 |
| **游戏活动** | 课堂游戏、互动活动设计 | ✅ 保留，适用于任何课程体系 |
| **教学原理** | 二语习得、艾宾浩斯曲线、螺旋上升等 | ✅ 保留，这些是底层理论 |

### 必须剔除的内容（逻辑冲突）

| 类型 | 说明 | 处理方式 |
|------|------|---------|
| **旧体系级别引用** | "Stage 2学生应该..."、"3H Book 1包含..." | ❌ 删除或替换为新体系级别 |
| **旧体系进阶逻辑** | "Stage 1→Stage 2→Stage 3..."的进阶路径 | ❌ 删除，使用新体系路径 |
| **旧教材内容引用** | "Hip Hip Hooray第X单元..."、"3H教材中..." | ❌ 删除具体教材引用 |
| **旧体系对标数据** | "3H Book 1含160个关键词"等具体数据 | ❌ 删除，需用新体系数据替换 |

## 五、萃取时的处理流程

**当遇到旧体系内容时，执行以下流程**：

```
发现旧体系内容
       │
       ▼
┌──────────────────┐
│ 判断内容类型     │
└──────────────────┘
       │
       ├─── 教学技术/方法论 ──→ ✅ 保留，标注 `#通用方法`
       │
       ├─── 话术/沟通技巧 ──→ ✅ 保留，去除旧体系级别引用
       │
       ├─── 课堂管理/游戏 ──→ ✅ 保留，标注适用级别范围
       │
       ├─── 旧体系级别逻辑 ──→ ❌ 删除或映射到新体系
       │
       └─── 旧教材具体内容 ──→ ❌ 删除，标注 `#已淘汰`
```

## 六、文档标注规范

**当保留旧体系中的通用内容时，使用以下标注**：

```markdown
> **来源说明**：本内容提取自旧体系资料，已剔除级别逻辑冲突，保留通用教学方法。
>
> `#通用方法` `#已脱敏`
```

**当发现需要清理的旧体系内容时，使用以下标注**：

```markdown
> ⚠️ **待清理**：本段落包含旧体系（Stage/3H）内容引用，需要：
> - [ ] 删除旧体系级别引用
> - [ ] 替换为新体系对应级别
> - [ ] 更新相关数据
>
> `#待清理` `#旧体系`
```
