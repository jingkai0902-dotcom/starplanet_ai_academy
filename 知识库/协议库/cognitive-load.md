# 认知负荷标签协议 V1.0

> 协议编号：7.16
> 创建日期：2026-01-24
> 适用场景：萃取游戏活动时

---

## 一、核心原则

**为每个游戏活动增加"认知负荷"标签，并自动关联游戏化教学模板，增加变体示例，支持智能备课系统。**

---

## 二、定义与价值

| 概念 | 定义 | 价值 |
|------|------|------|
| **认知负荷** | 学生完成活动所需的心智努力程度 | 支持活动合理排序 |
| **负荷匹配** | 根据教学环节选择合适负荷的活动 | 优化课堂节奏 |
| **变体关联** | 同一活动的不同难度版本 | 支持分层教学 |

---

## 三、认知负荷标签体系

| 标签 | 定义 | 适用环节 | 学生状态 |
|------|------|---------|---------|
| `#低负荷-热身` | 简单、轻松、激活状态 | Warm-up、课间过渡 | 刚进教室、注意力分散 |
| `#中负荷-操练` | 适度挑战、巩固练习 | Practice、Drill | 注意力集中、状态稳定 |
| `#高负荷-产出` | 高度挑战、综合运用 | Production、Assessment | 精力充沛、准备好挑战 |
| `#调节-放松` | 降低紧张、恢复精力 | 高强度活动后 | 疲劳、需要休息 |

## 五、认知负荷标签示例

```markdown
### 游戏：Whisper Game（传声筒）

**基本信息**
| 项目 | 内容 |
|------|------|
| **认知负荷** | `#中负荷-操练` |
| **适用环节** | Practice |
| **适用级别** | K3-G3 |
| **教具需求** | 单词/句子卡片 |
| **时间预估** | 8-10分钟 |

**操作步骤**
1. 分成两组，每组排成一列
2. 老师向排头展示卡片或耳语告诉句子
3. 排头学生将句子耳语传给身后的下一名学生
4. 依次向后传递
5. 最后一名学生大声说出完整句子
6. 最快且准确的队伍获胜

**变体示例**

| 变体 | 认知负荷 | 适用级别 | 调整说明 |
|------|---------|---------|---------|
| **简化版** | `#低负荷-热身` | K2-K3 | 只传单词，不传句子；允许看口型 |
| **标准版** | `#中负荷-操练` | G0-G1 | 传3-5词短句；标准耳语传递 |
| **进阶版** | `#高负荷-产出` | G2-G3 | 传7-10词长句；最后一人需写出句子 |
| **挑战版** | `#高负荷-产出` | G3 | 传两个句子；需按顺序复述 |

**关联模板**
- 游戏化教学模板：传递类游戏模板
- 相似游戏：Telephone Game、Message Relay

**课堂节奏建议**
```
Warm-up（低负荷）→ Whisper Game标准版（中负荷）→ 高强度产出（高负荷）→ 放松活动（调节）
```

`#执行层` `#中负荷-操练`
[Source ID: 游戏-Whisper-01]
```

## 六、课堂节奏设计建议

**标准45分钟课堂的认知负荷曲线：**

```
认知负荷
    ↑
高  │         ████
    │       ██    ██
中  │     ██        ██
    │   ██            ██
低  │ ██                ██
    └─────────────────────→ 时间
      0   10   20   30   45分钟

      热身  操练  产出  复习  结束
```

**推荐节奏**：
1. **0-5分钟**：低负荷热身，激活状态
2. **5-15分钟**：中负荷操练，巩固知识
3. **15-30分钟**：高负荷产出，综合运用
4. **30-40分钟**：中负荷复习，强化记忆
5. **40-45分钟**：低负荷收尾，总结放松
