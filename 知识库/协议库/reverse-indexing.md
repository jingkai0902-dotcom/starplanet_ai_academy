## 一、核心原则

**AI为每道教师考核题标注对应《教学标准化流程手册》或《课程体系全景图》的具���章节，实现精准推送"补课"素材。**

---

## 二、定义与价值

| 概念 | 定义 | 价值 |
|------|------|------|
| **反向索引** | 从考核题反向关联到知识源文档 | 支持精准补课 |
| **章节定位** | 精确到具体章节（如#SOP-3.2） | 提升学习效率 |
| **能力诊断** | 通过答题情况诊断能力短板 | 支持个性化培训 |

## 四、知识点反向索引示例

```markdown
### 题目：PPP教学模型的三个步骤依次是？

**选项**
- A. Practice - Presentation - Production
- B. Presentation - Production - Practice
- C. Presentation - Practice - Production
- D. Production - Practice - Presentation

**答案**：C

**解析**
PPP模型的正确顺序是：Presentation（呈现）→ Practice（操练）→ Production（产出）。这个顺序符合认知规律：先输入新知识，再通过练习巩固，最后自由产出。

---

**知识点反向索引**

| 索引项 | 内容 |
|--------|------|
| **知识点名称** | PPP教学模型 |
| **所属模块** | 教学方法 |
| **源文档** | 教学标准化流程手册 |
| **章节定位** | 管理层专区/1-工具模板库/教学标准化流程手册.md#1-1-ppp模型 |
| **能力维度** | 理论理解 |

**补课素材（答错时推送）**

**核心知识点**
PPP模型是语法和句型教学的核心方法，适用于G阶段的句型教学。

```
PPP教学模型
├── 1. Presentation（呈现）- 10-15分钟
│   ├── Lead-in: 情境导入
│   ├── Elicit: 引出句型
│   ├── Meaning Check: 确保理解含义
│   └── Pronunciation: 正音/打节奏
│
├── 2. Practice（操练）- 10-15分钟
│   ├── Substitution: 替换练习（机械操练）
│   └── Scaffolding: 搭梯子练习
│
└── 3. Production（产出）- 5-10分钟
    └── Contextual Application: 结合情境的任务
        （Survey, Mingle game等）
```

**操作要点**
1. Presentation阶段要确保学生理解含义，不只是机械跟读
2. Practice阶段要从机械操练逐步过渡到有意义的操练
3. Production阶段要给学生真实的语境，让他们自由运用

**常见错误**
| 错误 | 正确做法 |
|------|---------|
| 跳过Presentation直接Practice | 先确保学生理解，再进行操练 |
| Practice时间过长，Production不足 | 控制Practice在15分���内，留足Production时间 |
| Production只是重复Practice | Production要有新情境，让学生创造性运用 |

**延伸学习**
- 教学标准化流程手册.md#1-2-mpf模型
- 教学标准化流程手册.md#1-3-story五步法

`#测评项` `#反向索引`
[Source ID: 题库-教师-001]
```

## 五、能力诊断报告模板

**当教师在某类题目上错误率较高时，系统应生成能力诊断报告：**

```markdown
---
## 能力诊断报告

### 诊断信息

| 项目 | 内容 |
|------|------|
| **教师姓名** | [姓名] |
| **考核日期** | [日期] |
| **总分** | [分数] |
| **通过状态** | [通过/未通过] |

### 能力短板分析

| 模块 | 正确率 | 状态 | 补课建议 |
|------|--------|------|---------|
| 教学方法 | 60% | 🟡 需加强 | 重点学习PPP、M-P-F模型 |
| 课堂管理 | 80% | 🟢 达标 | - |
| Phonics教学 | 50% | 🔴 薄弱 | 系统学习Phonics教学指南 |
| 阅读教学 | 70% | 🟡 需加强 | 重点学习PWP模型 |

### 精准补课清单

**优先级1：Phonics教学（正确率50%）**
- [ ] 学习：Phonics教学完整指南.md（全文）
- [ ] 重点：K1-K3分阶段教学目标
- [ ] 练习：完成Phonics专项练习题（10道）
- [ ] 考核：1周后重新考核Phonics模块

**优先级2：教学方法（正确率60%）**
- [ ] 学习：教学标准化流程手册.md#核心教学模型
- [ ] 重点：PPP模型、M-P-F模型
- [ ] 练习：观摩优秀教师演课视频
- [ ] 考核：2周后重新考核教学方法模块

### 学习路径建议

```
第1周：Phonics教学
├── Day 1-2：阅读Phonics教学完整指南
├── Day 3-4：观摩K2 Phonics示范课
├── Day 5：完成专项练习题
└── Day 6-7：准备重新考核

第2周：教学方法
├── Day 1-2：阅读教学标准化流程手册
├── Day 3-4：观摩PPP模型示范课
├── Day 5：完成专项练习题
└── Day 6-7：准备重新考核
```

`#能力诊断` `#个性化培训`
```

- [ ] 每道题是否标注了知识点名称？

- [ ] 是否列出了常见错误？

- [ ] 是否使用了 `#SOP-X.X` 格式的章节标签？
