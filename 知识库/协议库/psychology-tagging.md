# 7.12 沟通心理学标签协议 (Communication Psychology Tagging Protocol)

> 来源：知识库萃取标准与执行规则.md V3.5
> **核心原则：AI在萃取话术时，必须识别并标注每段话术背后的心理动机，帮助新员工理解"为什么这么说"，同时为后续开发智能话术助手提供逻辑支撑。**

## 沟通心理学标签的定义与价值

| 概念 | 定义 | 价值 |
|------|------|------|
| **心理学标签** | 标注话术背后的心理学原理 | 帮助理解话术有效性 |
| **动机识别** | 识别话术试图触发的心理反应 | 支持话术优化 |
| **智能支撑** | 为AI话术助手提供逻辑基础 | 支持程序化调用 |

## 标准心理学标签体系

**以下为话术萃取时必须使用的心理学标签：**

| 标签 | 心理学原理 | 适用场景 | 示例话术 |
|------|-----------|---------|---------|
| `#缓解焦虑` | 降低认知失调，减少决策压力 | 家长担心效果、价格犹豫 | "这是正常的，很多家长一开始都有这个顾虑" |
| `#价值锚定` | 锚定效应，建立价值参照点 | 价格异议、竞品对比 | "您看，公立学校一年只教50个单词，我们是200个" |
| `#稀缺性建立` | 稀缺原理，制造紧迫感 | 促单、限时优惠 | "这个优惠只到本周日，名额也只剩3个了" |
| `#社会认同` | 从众心理，利用群体行为 | 建立信任、消除疑虑 | "我们班90%的家长都选择了续费" |
| `#权威背书` | 权威效应，借助专业认可 | 课程介绍、效果证明 | "这套课程是剑桥大学认证的" |
| `#损失规避` | 损失厌恶，强调不行动的代价 | 续费、升班 | "如果现在不续费，孩子可能会落后同龄人" |
| `#互惠原则` | 互惠心理，先给予再索取 | 转介绍、增值服务 | "我们免费送您一节测评课" |
| `#承诺一致` | 一致性原理，利用已有承诺 | 跟进、关单 | "您之前说最看重效果，我们的效果是..." |
| `#情感共鸣` | 共情效应，建立情感连接 | 异议处理、投诉应对 | "我完全理解您的感受，换做是我也会这样想" |
| `#具象化` | 具体化效应，用画面感增强说服力 | 效果描述、场景构建 | "想象一下，孩子上小学后英语课对他来说就是复习课" |
| `#对比效应` | 对比原理，通过对比突出优势 | 竞品对比、方案选择 | "线上课一年学的，可能还不如我们一个学期" |
| `#选择架构` | 框架效应，引导决策方向 | 报价、方案推荐 | "您看是选周六还是周日的班？" |

## 心理学标签使用规范

**话术萃取时，必须在话术末尾标注心理学标签：**

```markdown
#### 场景：家长说"价格太贵了"

**标准话术**
> "XX妈妈，我理解您的顾虑。其实很多家长一开始也有这个想法。但您有没有算过，孩子在励步学一年，词汇量能达到200个，而公立学校一年只教50个。平均下来，每个单词的学习成本其实比补习班还低。"

**心理学标签**
- `#缓解焦虑`：先认同家长感受，降低防御
- `#社会认同`：提及"很多家长"，利用从众心理
- `#价值锚定`：用词汇量对比建立价值参照
- `#具象化`：用"每个单词的成本"具象化价值

**底层逻辑**
1. 先共情（缓解焦虑）→ 2. 再正常化（社会认同）→ 3. 重构价值（价值锚定）→ 4. 具体化收益（具象化）

`#执行层` `#测评项` `#缓解焦虑` `#社会认同` `#价值锚定` `#具象化`
[Source ID: 销售话术-价格异议-01]
```

## 心理学路径图

**对于复杂话术，必须绘制心理学路径图：**

```
具象化（建立信任）
    ↓
价值锚定（建立期待）
    ↓
权威背书 + 损失规避（强化必要性）
    ↓
稀缺性 + 社会认同（制造紧迫感）
    ↓
选择架构（引导行动）
```

## 心理学标签检查清单

**每条话术萃取完成后，必须检查：**

- [ ] 是否识别了话术中的心理学原理？
- [ ] 是否标注了对应的心理学标签？
- [ ] 是否解释了每个标签的作用机制？
- [ ] 是否绘制了心理学路径图（复杂话术）？
- [ ] 标签是否使用了标准标签体系中的标签？
