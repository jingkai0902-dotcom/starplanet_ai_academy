# 全局关联基准协议 V1.0

> 协议编号：7.17
> 创建日期：2026-01-24
> 适用场景：生成级别相关文档时

---

## 一、核心原则

**《励步课程体系全景图》被定义为"全局关联的基准"，任何关于具体级别的萃取报告必须在开头引用全景图中的对应能力指标，实现"全息视角"。**

---

## 二、定义与价值

| 概念 | 定义 | 价值 |
|------|------|------|
| **全局基准** | 以《课程体系全景图》为所有级别文档的参照基准 | 确保内容一致性 |
| **能力锚定** | 每个级别文档必须锚定全景图中的能力指标 | 支持全局导航 |
| **全息视角** | 从全景图可以点击进入任何微观知识点 | 提升用户体验 |

---

## 三、全局关联引用格式

**任何关于具体级别的萃取报告/工作手册，必须在开头包含以下引用块：**

```markdown
---
## 全局定位（引用自《励步课程体系全景图》）

### 当前级别：[级别名称]

**在课程体系中的位置**
```
[上一级别] → 【当前级别】 → [下一级别]
```

**能力指标基准**（摘自全景图）

| 维度 | 目标能力 | 对标参照 |
|------|---------|---------|
| **听力** | [全景图中的听力目标] | [对标说明] |
| **口语** | [全景图中的口语目标] | [对标说明] |
| **阅读** | [全景图中的阅读目标] | [对标说明] |
| **书写** | [全景图中的书写目标] | [对标说明] |

**关键过渡期**
- 前置要求：[从上一级别需要达成的能力]
- 本级别重点：[本级别的核心突破点]
- 后续衔接：[为下一级别做的准备]

**全景图链接**：[决策层专区/励步课程体系全景图.md#[锚点]]

---
```

## 四、全局关联引用示例

```markdown
---
## 全局定位（引用自《励步课程体系全景图》）

### 当前级别：G1

**在课程体系中的位置**
```
K系列路径：K1 → K2 → K3 ─┐
                         ├→ 【G1】 → G2（时态突破）→ G3 → SS
G0入口路径：G0(20周) ────┘

注：K1、K2、G0均为零基础招生入口
```

**能力指标基准**（摘自全景图）

| 维度 | 目标能力 | 对标参照 |
|------|---------|---------|
| **听力** | 听懂100-150词的故事，抓取细节信息 | 超越公立小学1-2年级 |
| **口语** | 用完整句子描述图片，进行简单对话 | 对标剑桥Starters |
| **阅读** | 独立阅读100词文章，掌握三大阅读技能 | 超越公立小学2-3年级 |
| **书写** | 书写完整句子，简单段落仿写 | 超越公立小学2年级 |

**关键过渡期**
- 前置要求：完成K3或G0，具备CVC拼读能力，能说4-7词句子
- 本级别重点：从"Learn to Read"到"Read to Learn"的转变
- 后续衔接：为G2的时态学习和CLIL深化做准备

**全景图链接**：[决策层专区/励步课程体系全景图.md#g1]

---

## 一、G1级别教学指南

[正文内容...]
```

## 六、检查清单

**AI在生成级别相关文档时，必须检查：**

- [ ] 是否在开头包含了"全局定位"引用块？
- [ ] 引用的能力指标是否与《全景图》一致？
- [ ] 是否标注了关键过渡期信息？
- [ ] 是否提供了全景图链接（含锚点）？
- [ ] 《全景图》是否已更新全局关联索引表？
