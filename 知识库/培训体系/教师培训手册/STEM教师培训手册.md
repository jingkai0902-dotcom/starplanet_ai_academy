## 一、课程体系概述

### 1.1 分龄分阶设计

| 阶段 | 年龄 | 核心领域 | 认知特点 |
|------|------|----------|----------|
| 小班 | 3-4岁 | 生命科学、感官认知、基础数理 | 具象感知，以感官探索为主 |
| 中班 | 4-5岁 | 机械原理、建筑结构、编程启蒙 | 具象到抽象过渡 |
| 大班 | 5-6岁 | 复杂机械、地球空间、能源科学 | 抽象思维发展 |

### 1.2 模块划分

**小班四大模块**
- 1-1 认识我自己（人体科学）
- 1-2 动物王国（动物科学）
- 1-3 植物奥秘（植物科学）
- 1-4 数理逻辑与物理初探

**中班四大模块**
- 2-1 机械与工具（杠杆、滑轮、轮轴）
- 2-2 建筑与结构（互锁、拱形、三角稳定）
- 2-3 智能机械（传感器、编程启蒙）
- 2-4 物理科学（声光电热）

**大班四大模块**
- 3-1 复杂机械（齿轮、凸轮、连杆、棘轮）
- 3-2 地球与空间（火山、地震、四大发明）
- 3-3 能源科学（声光电磁、能量转换）
- 3-4 智能硬件（避障、巡线、机械臂）

### 1.3 课时安排

- 每节课时长：90分钟
- 建议分配：
  - 5E流程：70分钟
  - 收尾整理：10分钟
  - 弹性时间：10分钟

## 二、5E探究式教学法

### 2.1 5E流程概述

```
┌─────────────────────────────────────────────────────────────┐
│                      5E探究式教学法                          │
├─────────┬─────────┬─────────┬─────────┬─────────────────────┤
│ Engage  │ Explore │ Explain │Elaborate│     Evaluate        │
│  参与   │  探索   │  解释   │  拓展   │       评价          │
├─────────┼─────────┼─────────┼─────────┼─────────────────────┤
│ 5-10分钟│15-20分钟│10-15分钟│15-20分钟│     5-10分钟        │
└─────────┴─────────┴─────────┴─────────┴─────────────────────┘
```

### 2.2 各环节详解

#### Engage（参与）- 5-10分钟

**目标**：激发好奇心，引出主题

**标准操作**：
1. 播放动画或展示实物引入话题
2. 提出引导性问题
3. 连接学生已有经验

**话术示例**：
> "小朋友们，你们见过跷跷板吗？为什么一边坐重的人，另一边就会翘起来呢？今天我们来当小科学家，研究这个秘密！"

**禁忌**：
- ❌ 直接告诉答案
- ❌ 跳过引入直接开始搭建
- ❌ 引入时间过长（超过10分钟）

#### Explore（探索）- 15-20分钟

**目标**：动手实践，观察现象

**标准操作**：
1. 分发材料，说明任务
2. 让学生自主搭建/实验
3. 教师巡视指导，不直接干预
4. 鼓励学生记录观察结果

**话术示例**：
> "现在每个小组都有一套材料，请你们搭建一个跷跷板，试试看怎样才能让它平衡。遇到困难可以和小伙伴讨论。"

**禁忌**：
- ❌ 手把手教学生搭建
- ❌ 学生一遇到困难就帮忙解决
- ❌ 催促学生快点完成

#### Explain（解释）- 10-15分钟

**目标**：归纳现象，建立概念

**标准操作**：
1. 请学生分享发现
2. 引导归纳规律
3. 引入科学概念/术语
4. 用图示或模型解释原理

**话术示例**：
> "刚才小明发现，重的东西要放在靠近中间的位置才能平衡。这个中间的点叫做'支点'。谁能告诉我，支点是什么？"

**禁忌**：
- ❌ 只讲概念不联系实验
- ❌ 使用过于专业的术语
- ❌ 忽略学生的错误理解

#### Elaborate（拓展）- 15-20分钟

**目标**：应用知识，解决新问题

**标准操作**：
1. 提出新的挑战任务
2. 让学生应用所学知识
3. 鼓励创新和改进
4. 跨学科联系

**话术示例**：
> "现在我们来挑战一下！能不能用今天学的杠杆原理，设计一个投石机？看谁的投石机能把小球投得最远！"

**禁忌**：
- ❌ 拓展任务与主题无关
- ❌ 任务难度过高导致挫败
- ❌ 没有给足够的探索时间

#### Evaluate（评价）- 5-10分钟

**目标**：反思学习，评估理解

**标准操作**：
1. 请学生总结今天学到了什么
2. 展示优秀作品
3. 讨论失败的原因和改进方法
4. 布置延伸思考

**话术示例**：
> "今天我们学习了杠杆原理。谁能告诉我，杠杆有哪三个重要的部分？回家后，找找看家里有哪些东西用到了杠杆原理。"

**禁忌**：
- ❌ 跳过评价环节
- ❌ 只表扬成功，不讨论失败
- ❌ 评价变成批评

### 2.3 5E教学法核心原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 学生主体 | 教师是引导者，不是讲授者 | 让学生自己发现规律 |
| 问题驱动 | 用问题引导思考 | "为什么会这样？" |
| 允许失败 | 失败是学习的一部分 | "失败了没关系，我们来看看为什么" |
| 动手实践 | 做中学，学中做 | 每节课都有动手环节 |
| 联系生活 | 知识要能应用 | "生活中哪里用到了这个原理？" |

## 三、机械结构教学指南

### 3.1 简单机械教学

#### 杠杆（中班2-1、大班3-1）

**核心概念**：
- 支点、力点、重点
- 省力杠杆 vs 费力杠杆
- 杠杆原理：力 × 力臂 = 阻力 × 阻力臂

**教学话术**：
> "杠杆就像一个翘翘板。中间的支撑点叫'支点'，就像跷跷板的'腰'。你用力的地方叫'力点'，要抬起的东西在'重点'。"

**常见误区**：
| 误区 | 正确理解 |
|------|---------|
| 杠杆一定省力 | 费力杠杆省距离不省力 |
| 支点必须在中间 | 支点位置决定省力程度 |

**实验建议**：
1. 用尺子和橡皮做简易杠杆
2. 改变支点位置，观察变化
3. 用弹簧秤测量力的大小

#### 滑轮（中班2-1、大班3-1）

**核心概念**：
- 定滑轮：改变方向，不省力
- 动滑轮：省力一半，不改变方向
- 滑轮组：既省力又改变方向

**教学话术**：
> "定滑轮就像一个'方向盘'，你往下拉，东西往上升。动滑轮就像一个'小帮手'，帮你分担一半的力气。"

**实验建议**：
1. 用弹簧秤对比直接提和用滑轮提的力
2. 搭建简易起重机

#### 齿轮（中班2-2、大班3-1）

**核心概念**：
- 相邻齿轮转向相反
- 大齿轮带小齿轮：加速
- 小齿轮带大齿轮：减速

**教学话术**：
> "齿轮的牙齿像小手一样，你推我，我推你，就转起来了。大齿轮像大象走路，慢慢的；小齿轮像小老鼠，跑得快。"

**实验建议**：
1. 在齿轮上贴标记，观察转向
2. 数齿数，验证传动比

### 3.2 复杂机构教学（大班）

#### 连杆机构

**核心概念**：
- 将旋转运动转换为往复运动
- 曲柄摇杆机构

**教学话术**：
> "连杆就像'翻译官'，把转圈圈的运动翻译成前后动或上下动。"

#### 凸轮机构

**核心概念**：
- 凸轮形状决定运动规律
- 圆心凸轮、心形凸轮

**教学话术**：
> "凸轮就像一个'不规则的轮子'，它转的时候，上面的东西就会一高一低地跳。"

#### 棘轮机构

**核心概念**：
- 只允许单向转动
- 棘爪的锁定作用

**教学话术**：
> "棘爪就像一个'小卫士'，它说'只能往前走，不许往后退！'"

### 3.3 机械结构教学要点

| 要点 | 说明 |
|------|------|
| 先体验后讲解 | 让学生先动手，再总结原理 |
| 用类比解释 | 用生活中的例子解释抽象概念 |
| 强调观察 | 引导学生观察运动方向、速度变化 |
| 联系生活 | 找出生活中的应用实例 |

## 四、实验指导规范

### 4.1 实验前准备

**材料准备清单**：
- [ ] 检查材料数量是否充足
- [ ] 检查材料是否完好
- [ ] 准备备用材料
- [ ] 准备演示用材料

**场地准备**：
- [ ] 桌椅摆放适合小组活动
- [ ] 确保每组有足够空间
- [ ] 准备材料分发区

**安全检查**：
- [ ] 检查是否有尖锐部件
- [ ] 检查电子元件是否正常
- [ ] 准备急救箱

### 4.2 实验中指导

**巡视要点**：
1. 观察学生操作是否安全
2. 记录学生的发现和问题
3. 适时提问引导思考
4. 不要急于纠正"错误"

**介入时机**：
| 情况 | 处理方式 |
|------|---------|
| 安全问题 | 立即介入 |
| 完全卡住 | 提示性问题引导 |
| 方向偏离 | 温和引导回主题 |
| 有创新想法 | 鼓励尝试 |

**提问技巧**：
- "你观察到了什么？"
- "为什么会这样？"
- "如果改变...会怎样？"
- "你能想到其他方法吗？"

### 4.3 实验后整理

**材料整理**：
1. 指导学生分类整理材料
2. 检查材料是否完整
3. 清点并归位

**作品处理**：
- 拍照记录优秀作品
- 允许学生带走简单作品
- 复杂作品展示后拆解

## 五、课堂管理技巧

### 5.1 小班（3-4岁）管理

**特点**：
- 注意力时间短（5-10分钟）
- 需要更多具体指令
- 容易被新事物吸引

**策略**：
| 策略 | 示例 |
|------|------|
| 分段活动 | 每5-10分钟切换活动形式 |
| 具体指令 | "把红色积木放在蓝色积木上面" |
| 视觉提示 | 用图片展示步骤 |
| 即时反馈 | "做得真棒！" |

**常用口令**：
- "小眼睛看老师"
- "小手放膝盖"
- "1-2-3，坐坐好"

### 5.2 中班（4-5岁）管理

**特点**：
- 注意力时间延长（10-15分钟）
- 开始有合作意识
- 喜欢竞争和挑战

**策略**：
| 策略 | 示例 |
|------|------|
| 小组合作 | 2-3人一组完成任务 |
| 适度竞争 | "看哪个小组先完成" |
| 角色分配 | "你来搭建，他来测试" |
| 规则明确 | 提前说明规则和奖励 |

### 5.3 大班（5-6岁）管理

**特点**：
- 注意力时间较长（15-20分钟）
- 有较强的自主性
- 能进行简单的逻辑推理

**策略**：
| 策略 | 示例 |
|------|------|
| 任务驱动 | 给出明确的挑战目标 |
| 自主探索 | 减少干预，鼓励尝试 |
| 同伴学习 | 让学生互相教 |
| 反思总结 | 引导学生自己总结 |

### 5.4 常见课堂问题处理

| 问题 | 处理方式 |
|------|---------|
| 学生注意力分散 | 切换活动形式，用新奇事物吸引 |
| 学生争抢材料 | 明确分配规则，强调合作 |
| 学生遇到困难放弃 | 降低难度，分步引导 |
| 学生完成太快 | 提供进阶挑战任务 |
| 学生之间冲突 | 暂停活动，引导解决 |

## 六、常见问题与解决方案

### 6.1 教学问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 学生不感兴趣 | 引入不够吸引 | 用故事、动画、实物引入 |
| 学生听不懂 | 概念太抽象 | 用类比、实物演示 |
| 时间不够用 | 环节时间分配不当 | 提前演练，控制每环节时间 |
| 学生差异大 | 能力水平不同 | 分层任务，同伴互助 |

### 6.2 材料问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 材料不够 | 准备不足 | 提前清点，准备备用 |
| 材料损坏 | 使用不当 | 演示正确使用方法 |
| 材料丢失 | 整理不规范 | 建立材料管理制度 |

### 6.3 安全问题

| 问题 | 预防措施 |
|------|---------|
| 小零件误吞 | 3岁以下避免小零件，加强看护 |
| 尖锐部件划伤 | 检查材料，教导正确使用 |
| 电子元件触电 | 使用低压安全电源 |
