## 项目概述

斯坦星球知识库是一个企业级知识管理系统，通过NotebookLM直接处理原始PDF文档，利用AI进行知识萃取和问答。

### 核心流程

```
原始PDF/文档 → NotebookLM上传 → AI萃取 → 人工审核 → 知识库发布
```
