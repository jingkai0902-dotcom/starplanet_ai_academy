> **更新日期**：2026-02-10

## 进阶路线图

```
STEM小班              STEM中班            STEM大班            CODE硬件
认识机械        →     简单机械      →     复杂机构      →     机电一体
```

## 一、STEM小班阶段：认识机械（3-4岁）

### 1.1 学习特点

- 通过观察认识基本机械元件
- 感受"转动"、"移动"等基本运动
- 建立对机械的初步兴趣

### 1.2 课程出现

| 课程编号 | 课程名称 | 机械元素 | 学习目标 |
|----------|---------|---------|---------|
| STEM 1-1-01 | 旋转的风车 | 轮子 | 认识旋转 |
| STEM 1-1-02 | 小车跑跑 | 轮轴 | 认识滚动 |
| STEM 1-1-03 | 跷跷板 | 杠杆 | 认识平衡 |

### 1.3 认知目标

- 能说出"转"、"动"、"平衡"等词
- 能观察并描述简单运动
- 对机械产生好奇心

### 1.4 教学话术

> "看，风车在转！你能让它转得更快吗？"
> "小车的轮子是圆的，所以能滚动。"

## 二、STEM中班阶段：简单机械（4-5岁）

### 2.1 学习特点

- 理解简单机械的基本原理
- 能动手搭建简单机械装置
- 初步理解"省力"概念

### 2.2 课程出现

| 课程编号 | 课程名称 | 机械类型 | 核心概念 |
|----------|---------|---------|---------|
| STEM 2-1-03 | 快乐跷跷板 | 杠杆 | 平衡、支点 |
| STEM 2-1-04 | 神奇的投石机 | 杠杆 | 费力杠杆 |
| STEM 2-1-07 | 有趣的滑轮 | 滑轮 | 改变方向 |
| STEM 2-1-08 | 建筑工地 | 履带/轮子 | 摩擦力 |
| STEM 2-2-10 | 荷兰风车 | 齿轮 | 传动 |

### 2.3 核心概念

#### 杠杆三要素（初步）

```
支点：跷跷板中间的支撑点
力点：你坐的地方
重点：小伙伴坐的地方
```

#### 滑轮（初步）

```
定滑轮：固定不动，改变方向
"往下拉，东西往上升"
```

#### 齿轮（初步）

```
齿轮啮合：牙齿咬在一起
"你转我也转"
```

### 2.4 教学话术

> "跷跷板的中间叫'支点'，就像跷跷板的'腰'。"
> "滑轮像个'方向盘'，你往下拉，东西往上升。"
> "齿轮的牙齿像小手，你推我，我推你。"

## 三、STEM大班阶段：复杂机构（5-6岁）

### 3.1 学习特点

- 理解复杂机械原理
- 能分析机构的运动规律
- 能设计简单的机械组合

### 3.2 课程出现

| 课程编号 | 课程名称 | 机械类型 | 核心概念 |
|----------|---------|---------|---------|
| STEM 3-1-01 | 杠杆与平衡 | 杠杆 | 三要素、省力原理 |
| STEM 3-1-02 | 齿轮传动 | 齿轮 | 传动比、转向规律 |
| STEM 3-1-03 | 轮轴原理 | 轮轴 | 旋转的杠杆 |
| STEM 3-1-04 | 滑轮系统 | 滑轮 | 滑轮组、省力 |
| STEM 3-1-05 | 斜面与螺旋 | 斜面 | 省力必费距离 |
| STEM 3-1-09 | 连杆机构 | 连杆 | 运动转换 |
| STEM 3-1-10 | 凸轮机构 | 凸轮 | 运动规律设计 |
| STEM 3-1-11 | 棘轮与棘爪 | 棘轮 | 单向转动 |

### 3.3 核心概念

#### 杠杆原理（深化）

```
力 × 力��� = 阻力 × 阻力臂

省力杠杆：力臂 > 阻力臂
费力杠杆：力臂 < 阻力臂
等臂杠杆：力臂 = 阻力臂
```

#### 齿轮传动比

```
传动比 = 从动轮齿数 / 主动轮齿数

大齿轮带小齿轮：加速
小齿轮带大齿轮：减速
```

#### 运动转换

```
连杆：圆周运动 ↔ 往复运动
凸轮：旋转 → 任意规律运动
```

### 3.4 教学话术

> "如果你想用很小的力气抬起很重的东西，应该把支点靠近谁？"
> "大齿轮像大象走路，慢慢的；小齿轮像小老鼠，跑得快。"
> "连杆就像'翻译官'，把转圈圈的运动翻译成前后动。"

## 四、CODE硬件阶段：机电一体（6-10岁）

### 4.1 学习特点

- 机械结构与电子控制结合
- 传感器驱动的智能机械
- 编程控制机械运动

### 4.2 课程出现

| 课程编号 | 课程名称 | 机械+电子 | 核心概念 |
|----------|---------|---------|---------|
| CODE1-1-01 | 旋转的小鸟 | 齿轮+马达 | 电机驱动 |
| CODE1-1-02 | 高低传送带 | 皮带+马达 | 传送系统 |
| CODE2-1-02 | 避障小车 | 轮子+传感器 | 智能控制 |
| STEM 3-4-05 | 遥控机械臂 | 连杆+舵机 | 多自由度 |
| STEM 3-4-09 | 疯狂娃娃机 | 龙门架+电机 | 二维运动 |

### 4.3 核心概念

#### 电机驱动

```
马达 → 齿轮减速 → 输出
       ↓
    控制转速和力矩
```

#### 舵机控制

```
舵机 = 马达 + 齿轮 + 控制电路

输入：角度信号（0-180°）
输出：精确角度位置
```

#### 传感器反馈

```
传感器 → 控制器 → 执行器
  ↓         ↓         ↓
感知环境   做决策   机械动作
```

### 4.4 教学话术

> "舵机就像一个'听话的小电机'，你说转多少度，它就转多少度。"
> "传感器是机器的'眼睛'，让机器能'看到'周围的环境。"

## 五、机械结构知识图谱

### 5.1 简单机械

```
简单机械
├── 杠杆
│   ├── 省力杠杆（撬棍）
│   ├── 费力杠杆（筷子）
│   └── 等臂杠杆（天平）
├── 滑轮
│   ├── 定滑轮（改变方向）
│   ├── 动滑轮（省力）
│   └── 滑轮组（组合）
├── 轮轴
│   └── 旋转的杠杆
└── 斜面
    ├── 斜面（楼梯）
    └── 螺旋（螺丝）
```

### 5.2 传动机构

```
传动机构
├── 齿轮传动
│   ├── 直齿轮
│   ├── 锥齿轮
│   └── 蜗轮蜗杆
├── 皮带传动
│   ├── 平皮带
│   └── V带
└── 链条传动
    └── 链轮链条
```

### 5.3 运动转换

```
运动转换
├── 连杆机构
│   ├── 曲柄摇杆
│   ├── 曲柄滑块
│   └── 双曲柄
├── 凸轮机构
│   ├── 圆心凸轮
│   ├── 心形凸轮
│   └── 盘形凸轮
└── 槽轮机构
    └── 间歇运动
```

### 5.4 控制机构

```
控制机构
├── 棘轮机构
│   └── 单向转动
├── 离合器
│   └── 接合/分离
└── 制动器
    └── 减速/停止
```

## 七、教学建议

### 7.1 衔接策略

| 衔接点 | 策略 |
|--------|------|
| 小班→中班 | 从"观察"到"动手搭建" |
| 中班→大班 | 从"单一机械"到"机构组合" |
| 大班→CODE | 从"手动"到"电机驱动" |

### 7.2 常见教学难点

| 难点 | 解决方案 |
|------|---------|
| 杠杆三要素 | 用"腰、手、重物"类比 |
| 齿轮转向 | 用手指模拟啮合 |
| 传动比计算 | 用实物数齿数 |
| 连杆运动 | 用动画演示 |
| 凸轮规律 | 用慢动作视频 |
