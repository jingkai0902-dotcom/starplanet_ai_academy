## 什么是螺旋进阶？

螺旋进阶是指同一个核心概念在不同年龄段、不同课程体系中反复出现，每次出��都在前一次的基础上加深理解、扩展应用。

```
                    ┌─────────────────┐
                    │  Python高级应用  │  12岁+
                    │  OOP、AI、算法   │
                    └────────┬────────┘
                             │
                    ┌────────┴────────┐
                    │  Python基础      │  10-12岁

                    └────────┬────────┘
                             │
                    ┌────────┴────────┐
                    │  CODE图形化编程   │  6-10岁
                    │  积木、逻辑、硬件 │
                    └────────┬────────┘
                             │
                    ┌────────┴────────┐

                    │  观察、搭建、实验 │
                    └─────────────────┘
```

## 跨语言概念映射

### Scratch → Python 核心映射

| Scratch积木 | Python语法 | 概念 |
|------------|-----------|------|
| 重复10次 | `for i in range(10):` | 固定循环 |
| 重复执行 | `while True:` | 无限循环 |
| 重复直到<条件> | `while not condition:` | 条件循环 |
| 如果<条件>那么 | `if condition:` | 单分支 |
| 如果<条件>那么...否则 | `if...else:` | 双分支 |
| 设定[变量]为(值) | `variable = value` | 变量赋值 |
| 将[变量]改变(值) | `variable += value` | 变量修改 |
| 添加(项)到[列表] | `list.append(item)` | 列表添加 |
| 定义[自制积木] | `def function():` | 函数定义 |
| 广播[消息] | 事件/回调 | 消息传递 |
| 克隆自己 | 类实例化 | 对象创建 |

## 教学建议

### 螺旋进阶的教学原则

1. **复现强化**：同一概念多次出现，每次加深理解
2. **渐进复杂**：从简单到复杂，从具体到抽象
3. **迁移应用**：在新情境中应用已学概念
4. **关联建构**：建立新旧知识的联系

### 跨课程衔接要点

| 衔接点 | 前置课程 | 后续课程 | 衔接策略 |
|--------|---------|---------|---------|
| STEM→CODE | STEM大班 | CODE1 | 从实物到虚拟，保持动手习惯 |
| CODE1→CODE2 | CODE1 | CODE2 | 从纯软件到软硬结合 |
| CODE→Python | CODE2/3 | PYAI L1 | 从积木到代码，概念映射 |
| PYAI L1→L2 | PYAI L1 | PYAI L2 | 从基础到进阶，项目驱动 |
