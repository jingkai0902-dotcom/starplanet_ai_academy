## 二、CODE1阶段：变量（6-8岁）

### 2.1 概念深化

从具体的"计数"过渡到抽象的"变量"，理解变量是"存储数据的盒子"。

### 2.2 课程出现

| 课程编号 | 课程名称 | 变量类型 | 具体内容 |
|----------|---------|---------|---------|
| CODE1-3-10 | 变量基础 | 数字变量 | 设定[得分]为(0) |
| CODE1-4-02 | 随机数 | 随机变量 | 随机数(1到10) |
| CODE1-4-03 | 计分系统 | 变量应用 | 得分增减 |

### 2.3 Scratch积木

```scratch
// 创建变量
设定 [得分] 为 (0)

// 修改变量
将 [得分] 改变 (10)

// 使用变量
如果 <(得分) > (100)> 那么
    说 [你赢了！]

// 随机数
设定 [随机位置] 为 (随机取数 (1) 到 (10))
```

### 2.4 教学话术

> "变量就像一个盒子，可以放东西进去。盒子有名字（变量名），里面放的东西（值）可以换。得分就是一个变量，开始是0，吃到苹果就加10。"

### 2.5 常见错误

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| ❌ 变量名不清晰 | 代码难懂 | 用有意义的名字如"得分"、"生命" |
| ❌ 忘记初始化 | 值不确定 | 程序开始时设定初始值 |

// 创建列表

## 四、PYAI L1阶段：列表与字典（10-12岁）

### 4.1 概念转化

从图形化列表过渡到Python的列表和字典。

### 4.2 课程出现

| 课程编号 | 课程名称 | 数据类型 | 具体内容 |
|----------|---------|---------|---------|
| PYAI 1-3-1 | 列表基础 | 列表 | `list = []` |
| PYAI 1-2-9 | 字典 | 字典 | `dict = {}` |
| PYAI 1-3-6 | 二维列表 | 嵌套列表 | `board[row][col]` |

### 4.3 Python代码

```python
# 列表
fruits = ['apple', 'banana', 'cherry']
fruits.append('orange')  # 添加
fruits.remove('banana')  # 删除
print(fruits[0])         # 访问：apple
print(len(fruits))       # 长度：3

# 列表遍历
for fruit in fruits:
    print(fruit)

# 字典
student = {
    'name': '小明',
    'age': 12,
    'score': 95
}
print(student['name'])   # 访问：小明
student['grade'] = 'A'   # 添加键值对

# 二维列表（棋盘）
board = [
    [0, 0, 0],
    [0, 0, 0],
    [0, 0, 0]
]
board[1][1] = 1  # 中心位置设为1
```

### 4.4 Scratch → Python 映射

| Scratch积木 | Python语法 |
|------------|-----------|
| 添加(项)到[列表] | `list.append(item)` |
| 删除(列表的第n项) | `list.pop(n-1)` 或 `del list[n-1]` |
| 列表的第n项 | `list[n-1]` |
| 列表的项目数 | `len(list)` |

### 4.5 列表 vs 字典

| 特性 | 列表 | 字典 |
|------|------|------|
| 访问方式 | 索引（数字） | 键（字符串） |
| 顺序 | 有序 | 无序（Python 3.7+有序） |
| 用途 | 存储同类数据 | 存储属性 |
| 示例 | `[1, 2, 3]` | `{'a': 1, 'b': 2}` |

### 4.6 教学话术

> "列表用数字找东西，像排队：第1个、第2个...字典用名字找东西，像通讯录：找'小明'的电话。"

## 五、PYAI L2阶段：类属性（12岁+）

### 5.1 概念升华

从独立的数据结构升级到面向对象的类属性。

### 5.2 课程出现

| 课程编号 | 课程名称 | 数据类型 | 具体内容 |
|----------|---------|---------|---------|
| PYAI 2-2-1 | 类基础 | 类属性 | `self.name` |
| PYAI 2-2-2 | 类方法 | 实例方法 | `def method(self):` |

### 5.3 Python代码

```python
class Player:
    def __init__(self, name):
        # 实例属性
        self.name = name
        self.health = 100
        self.score = 0
        self.inventory = []  # 列表作为属性

    def take_damage(self, amount):
        self.health -= amount
        if self.health <= 0:
            self.die()

    def collect_item(self, item):
        self.inventory.append(item)
        self.score += 10

    def die(self):
        print(f"{self.name} 死亡了！")

# 使用
player1 = Player("小明")
player1.take_damage(30)
player1.collect_item("金币")
print(player1.health)     # 70
print(player1.inventory)  # ['金币']
```

### 5.4 数据组织演进

```
变量          →    列表          →    字典          →    类
score = 0         scores = []       player = {}       class Player:
                                    player['score']     self.score
单个值            多个同类值         多个不同属性        属性+方法
```

### 5.5 教学话术

> "类就像一个模板，定义了'玩家'应该有什么属性（名字、血量、得分）和能做什么（受伤、捡东西）。每个玩家都是这个模板的一个实例。"
