## 一、棘轮机构

### 1.1 基本概念

棘轮机构是一种单向运动机构，只允许轮子向一个方向转动。

```
棘轮机构：

        棘爪
         ↓
    ┌────▼────┐
    │  ╱╲╱╲   │
    │ ╱    ╲  │ ← 棘轮（有齿）
    │╱   ○  ╲ │
    │╲      ╱ │
    │ ╲    ╱  │
    │  ╲╱╲╱   │
    └─────────┘

棘爪卡住棘轮的齿，只允许单向转动
```

### 1.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 具体内容 |
|----------|---------|------|---------|
| STEM 3-1-11 | 棘轮与棘爪 | 5-6岁 | 单向转动原理 |

### 1.3 棘轮工作原理

```
正向转动（允许）：

    棘爪
     ↓
    ╱╲
   ╱  ╲  → 棘爪滑过齿顶
  ╱    ╲
 ╱──────╲
    ↻ 转动方向

反向转动（阻止）：

    棘爪
     ↓
    ╱╲
   ╱  ╲  ← 棘爪卡住齿根
  ╱    ╲
 ╱──────╲
    ↺ 被阻止
```

### 1.4 棘轮类型

#### 外棘轮

```
外棘轮：

    ┌─────────┐
    │ ╱╲╱╲╱╲  │ ← 齿在外圈
    │╱      ╲ │
    │   ○    │
    │╲      ╱ │
    │ ╲╱╲╱╲╱  │
    └─────────┘
         ↑
        棘爪在外侧
```

#### 内棘轮

```
内棘轮：

    ┌─────────┐
    │         │
    │  ╱╲╱╲   │ ← 齿在内圈
    │ ╱    ╲  │
    │╱  ○   ╲ │
    │ ╲    ╱  │
    │  ╲╱╲╱   │
    └─────────┘
         ↑
        棘爪在内侧
```

### 1.5 教学话术

> "棘爪就像一个'小卫士'，它说'只能往前走，不许往后退！'听到'咔哒咔哒'的声音，就是棘爪在一个齿一个齿地跳过去。"

### 1.6 生活实例

| 实例 | 功能 | 说明 |
|------|------|------|
| 扳手 | 单向拧紧 | 回程不松动 |
| 卷尺 | 单向收回 | 拉出后锁定 |
| 自行车飞轮 | 单向驱动 | 倒踩不倒退 |
| 钓鱼竿卷线器 | 单向收线 | 防止线松脱 |
| 手表上弦 | 单向上弦 | 防止发条松开 |

### 1.7 实验活动

#### 棘轮单向实验

```
材料：棘轮模型

步骤：
1. 正向转动棘轮，观察棘爪滑过齿顶
2. 听"咔哒咔哒"的声音
3. 尝试反向转动，观察棘爪卡住
4. 记录现象

结论：棘轮只能单向转动
```

## 二、离合器

### 2.1 基本概念

离合器是一种可以接合或分离两个轴的机构，用于控制动力传递。

```
离合器原理：

接合状态：
    主动轴 ═══╪═══ 从动轴
              │
           离合器
           （接合）

分离状态：
    主动轴 ═══   ═══ 从动轴
              │
           离合器
           （分离）
```

### 2.2 离合器类型

#### 摩擦离合器

```
摩擦离合器：

    主动盘     从动盘
       │         │
       ▼         ▼
    ┌─────┐ ┌─────┐
    │     │═│     │ ← 摩擦面接触
    └─────┘ └─────┘

通过摩擦力传递动力
应用：汽车离合器
```

#### 牙嵌离合器

```
牙嵌离合器：

    主动轴     从动轴
       │         │
       ▼         ▼
    ┌─────┐ ┌─────┐
    │╱╲╱╲ │═│╱╲╱╲ │ ← 齿啮合
    └─────┘ └─────┘

通过齿的啮合传递动力
应用：变速箱
```

### 2.3 教学话术

> "离合器就像一个'开关'，可以让两个轴连在一起转，也可以让它们分开。汽车换挡的时候，就要先踩离合器，让发动机和轮子分开。"

## 三、制动机构

### 3.1 基本概念

制动机构用于减速或停止运动部件。

```
制动原理：

    运动部件
        │
        ▼
    ┌───────┐
    │       │ ← 制动器
    │   ○   │   （产生摩擦力）
    │       │
    └───────┘

通过摩擦力消耗动能
```

### 3.2 制动类型

#### 鼓式制动

```
鼓式制动：

    ┌─────────┐
    │         │
    │  ┌───┐  │ ← 制动蹄
    │  │ ○ │  │ ← 制动鼓
    │  └───┘  │
    │         │
    └─────────┘

制动蹄向外张开，压住制动鼓
```

#### 盘式制动

```
盘式制动：

        制动钳
           │
           ▼
    ┌───┐ ═══ ┌───┐
    │   │     │   │ ← 制动盘
    └───┘     └───┘
           │
        制动片

制动片夹住制动盘
```

### 3.3 教学话术

> "制动器就像'刹车'，用摩擦力让转动的东西慢下来或停下来。自行车的刹车就是用橡皮块夹住车轮。"

### 3.4 生活实例

| 实例 | 制动类型 | 说明 |
|------|---------|------|
| 自行车 | 闸皮制动 | 夹住车圈 |
| 汽车 | 盘式/鼓式 | 液压制动 |
| 电梯 | 电磁制动 | 断电制动 |
| 起重机 | 带式制动 | 摩擦带 |

## 四、超越离合器

### 4.1 基本概念

超越离合器是一种单向离合器，只在一个方向传递动力。

```
超越离合器：

正向传动（接合）：
    主动轴 ═══╪═══ 从动轴
              │
           楔块卡紧

反向超越（分离）：
    主动轴 ═══   ═══ 从动轴
              │
           楔块松开
```

### 4.2 工作原理

```
楔块式超越离合器：

    ┌─────────────┐
    │    ╱╲      │
    │   ╱  ╲     │ ← 楔块
    │  ╱    ╲    │
    │ ╱  ○   ╲   │ ← 内圈
    │╱        ╲  │
    └─────────────┘ ← 外圈

正向：楔块卡紧，传递动力
反向：楔块松开，自由转动
```

### 4.3 生活实例

| 实例 | 功能 |
|------|------|
| 自行车飞轮 | 踩踏传动，滑行自由 |
| 电动工具 | 防止反转 |
| 自动扶梯 | 防止倒转 |

## 五、控制机构对比

| 机构 | 功能 | 控制方式 | 应用 |
|------|------|---------|------|
| 棘轮 | 单向转动 | 机械锁定 | 扳手、卷尺 |
| 离合器 | 接合/分离 | 人工控制 | 汽车变速 |
| 制动器 | 减速/停止 | 人工控制 | 刹车 |
| 超越离合器 | 单向传动 | 自动 | 自行车飞轮 |

## 七、常见错误与禁忌

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| ❌ 棘爪弹簧太弱 | 跳齿 | 更换弹簧 |
| ❌ 离合器打滑 | 动力损失 | 调整压紧力 |
| ❌ 制动片磨损 | 制动失效 | 及时更换 |
| ❌ 超越离合器卡死 | 无法超越 | 清洁润滑 |

## 八、教学建议

### 8.1 教学顺序

1. **听声音**：棘轮的"咔哒"声
2. **看现象**：单向转动演示
3. **找原因**：棘爪卡住齿
4. **联系生活**：扳手、卷尺

### 8.2 常见教学难点

| 难点 | 解决方案 |
|------|---------|
| 棘轮原理 | 用放大模型演示 |
| 离合器理解 | 用"开关"类比 |
| 制动原理 | 用自行车刹车演示 |
