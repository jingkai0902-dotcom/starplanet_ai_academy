## 一、简单机械

### 1.1 杠杆

#### 跨课程出现

| 课程编号 | 课程名称 | 杠杆类型 | 具体内容 |
|----------|---------|---------|---------|
| STEM 2-1-03 | 快乐跷跷板 | 等臂杠杆 | 平衡原理 |
| STEM 2-1-04 | 神奇的投石机 | 费力杠杆 | 省距离 |
| STEM 3-1-01 | 杠杆与平衡 | 省力杠杆 | 三要素 |
| STEM 3-2-06 | 古代投石机 | 费力杠杆 | 弹性势能 |

#### 核心概念

```
杠杆三要素：
┌─────────────────────────────────────┐
│                                     │
│    力点(F)      支点(O)      重点(W) │
│      ↓           △           ↓     │
│  ════════════════════════════════   │
│      ←── 力臂 ──→←── 阻力臂 ──→     │
│                                     │
└─────────────────────────────────────┘

杠杆原理：力 × 力臂 = 阻力 × 阻力臂
```

#### 杠杆分类

| 类型 | 特点 | 生活实例 |
|------|------|---------|
| 省力杠杆 | 力臂 > 阻力臂 | 撬棍、开瓶器 |
| 费力杠杆 | 力臂 < 阻力臂 | 筷子、钓鱼竿 |
| 等臂杠杆 | 力臂 = 阻力臂 | 天平、跷跷板 |

### 1.2 滑轮

#### 跨课程出现

| 课程编号 | 课程名称 | 滑轮类型 | 具体内容 |
|----------|---------|---------|---------|
| STEM 2-1-07 | 有趣的滑轮 | 定滑轮 | 改变方向 |
| STEM 3-1-04 | 滑轮系统 | 滑轮组 | 省力原理 |

#### 核心概念

```
定滑轮：                    动滑轮：
  ┌───┐                      ┌───┐
  │ ○ │ ← 固定               │ ○ │ ← 随物体移动
  └─┬─┘                      └─┬─┘
    │                          │
    ↓ 拉力                     ↓ 拉力 = 重力/2

特点：改变方向，不省力        特点：省力一半，不改变方向
```

### 1.3 轮轴

#### 跨课程出现

| 课程编号 | 课程名称 | 具体内容 |
|----------|---------|---------|
| STEM 3-1-03 | 轮轴原理 | 井轱辘、方向盘 |

#### 核心概念

```
轮轴 = 旋转的杠杆

      ┌─────────┐
      │    轮    │  ← 大轮（长力臂）
      │  ┌───┐  │
      │  │轴 │  │  ← 小轴（短力臂）
      │  └───┘  │
      └─────────┘

轮大轴小 → 省力
轮小轴大 → 省距离
```

## 二、传动机构

### 2.1 齿轮传动

#### 跨课程出现

| 课程编号 | 课程名称 | 齿轮类型 | 具体内容 |
|----------|---------|---------|---------|
| STEM 2-2-10 | 荷兰风车 | 齿轮组 | 风能传动 |
| STEM 3-1-02 | 齿轮传动 | 大小齿轮 | 传动比 |

#### 核心概念

```
齿轮啮合规律：

  ┌───┐     ┌─┐
  │ ↻ │ ──→ │↺│
  └───┘     └─┘
  大齿轮     小齿轮
  转得慢     转得快

相邻齿轮：转向相反
隔一个齿轮：转向相同
```

#### 传动比

```
传动比 = 从动轮齿数 / 主动轮齿数

例：大齿轮40齿，小齿轮10齿
传动比 = 40/10 = 4
大齿轮转1圈，小齿轮转4圈
```

## 六、常见错误与禁忌

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| ❌ 齿轮不啮合 | 无法传动 | 确保齿轮牙齿咬合 |
| ❌ 杠杆支点不稳 | 无法平衡 | 固定支点位置 |
| ❌ 滑轮绳子打滑 | 无法提升 | 绳子绑紧在滑轮槽内 |
| ❌ 连杆长度不对 | 卡死 | 按设计图纸搭建 |

## 七、教学建议

### 7.1 认知进阶路线

```
观察现象 → 动手搭建 → 理解原理 → 应用设计
   ↓           ↓           ↓           ↓
"它在转"    "我做出来了"  "因为..."   "我能改进"
```

### 7.2 常见教学难点

| 难点 | 解决方案 |
|------|---------|
| 杠杆三要素混淆 | 用"腰、手、重物"类比 |
| 齿轮转向规律 | 用手指模拟齿轮啮合 |
| 滑轮省力原理 | 实际称重对比 |
| 凸轮运动规律 | 用动画或实物演示 |
