## 一、齿轮传动

### 1.1 基本概念

齿轮是边缘有齿的轮子，通过齿的啮合传递运动和力。

```
齿轮啮合：

  ┌───────┐     ┌───┐
  │       │     │   │
  │   ↻   │ ──→ │ ↺ │
  │       │     │   │
  └───────┘     └───┘
   大齿轮        小齿轮
   转得慢        转得快
   力矩大        力矩小
```

### 1.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 齿轮类型 | 具体内容 |
|----------|---------|------|---------|---------|
| STEM 2-2-10 | 荷兰风车 | 4-5岁 | 齿轮组 | 风能传动 |
| STEM 3-1-02 | 齿轮传动 | 5-6岁 | 大小齿轮 | 传动比 |

### 1.3 齿轮啮合规律

#### 转向规律

```
两个齿轮啮合：转向相反

  ┌───┐     ┌───┐
  │ ↻ │ ──→ │ ↺ │
  └───┘     └───┘
  顺时针     逆时针

三个齿轮串联：首尾同向

  ┌───┐     ┌───┐     ┌───┐
  │ ↻ │ ──→ │ ↺ │ ──→ │ ↻ │
  └───┘     └───┘     └───┘
  顺时针     逆时针     顺时针

规律：相邻齿轮转向相反，隔一个齿轮转向相同
```

#### 速度规律

```
大齿轮带动小齿轮：加速

  ┌───────┐     ┌───┐
  │ 40齿  │ ──→ │10齿│
  │ 1圈   │     │4圈 │
  └───────┘     └───┘

小齿轮带动大齿轮：减速

  ┌───┐     ┌───────┐
  │10齿│ ──→ │ 40齿  │
  │4圈 │     │ 1圈   │
  └───┘     └───────┘
```

### 1.4 传动比

```
传动比 = 从动轮齿数 / 主动轮齿数
       = 主动轮转速 / 从动轮转速

例：主动轮20齿，从动轮40齿
传动比 = 40/20 = 2
主动轮转2圈，从动轮转1圈（减速）

例：主动轮40齿，从动轮10齿
传动比 = 10/40 = 0.25
主动轮转1圈，从动轮转4圈（加速）
```

### 1.5 齿轮类型

| 类型 | 特点 | 应用 |
|------|------|------|
| 直齿轮 | 齿平行于轴 | 最常见 |
| 斜齿轮 | 齿倾斜 | 更平稳 |
| 锥齿轮 | 改变轴向 | 90°传动 |
| 蜗轮蜗杆 | 大传动比 | 减速器 |
| 齿条 | 直线齿 | 直线运动 |

```
锥齿轮（改变方向）：

      │
    ┌─┴─┐
    │ ╲ │
    └───┘
       ╲
        ╲
      ┌──╲──┐
      │    │
      └────┘
         │

蜗轮蜗杆（大减速比）：

    ═══╪═══  蜗杆
       │
    ┌──┴──┐
    │     │  蜗轮
    └─────┘
```

### 1.6 教学话术

**中班（4-5岁）**：
> "齿轮的牙齿像小手一样，你推我，我推你，就转起来了。看，这个齿轮往这边转，旁边的齿轮就往那边转！"

**大班（5-6岁）**：
> "大齿轮像大象走路，慢慢的；小齿轮像小老鼠，跑得快。大齿轮转一圈，小齿轮要转好几圈呢！"

### 1.7 实验活动

#### 齿轮转向实验

```
材料：齿轮组、标记贴纸

步骤：
1. 在每个齿轮上贴一个标记
2. 转动第一个齿轮
3. 观察其他齿轮的转向
4. 记录规律

结论：相邻齿轮转向相反
```

#### 齿轮速度实验

```
材料：大小齿轮、计数器

步骤：
1. 大齿轮（40齿）带动小齿轮（10齿）
2. 大齿轮转1圈，数小齿轮转几圈
3. 交换主从关系，重复实验
4. 记录数据

结论：齿数比 = 转速反比
```

## 二、皮带传动

### 2.1 基本概念

皮带传动通过皮带连接两个轮子，传递运动和力。

```
皮带传动：

  ┌───────┐           ┌───┐
  │       │═══════════│   │
  │   ↻   │           │ ↻ │
  │       │═══════════│   │
  └───────┘           └───┘
   主动轮              从动轮
   （大）              （小）

特点：两轮转向相同（与齿轮相反）
```

### 2.2 皮带传动特点

| 特点 | 说明 |
|------|------|
| 转向相同 | 两轮同向旋转 |
| 可远距离传动 | 两轮可以相距较远 |
| 有打滑保护 | 过载时皮带打滑��保护机器 |
| 传动比不精确 | 有滑动损失 |

### 2.3 交叉皮带

```
交叉皮带传动：

  ┌───────┐     ╲     ┌───┐
  │       │      ╲    │   │
  │   ↻   │       ╳   │ ↺ │
  │       │      ╱    │   │
  └───────┘     ╱     └───┘

特点：两轮转向相反
```

### 2.4 教学话术

> "皮带就像一条'传送带'，把一个轮子的转动传给另一个轮子。和齿轮不一样，皮带连接的两个轮子转向是一样的！"

## 三、链条传动

### 3.1 基本概念

链条传动通过链条连接两个链轮，传递运动和力。

```
链条传动：

  ┌───────┐           ┌───┐
  │ ○ ○ ○ │═○═○═○═○═○═│○ ○│
  │   ↻   │           │ ↻ │
  │ ○ ○ ○ │═○═○═○═○═○═│○ ○│
  └───────┘           └───┘
   主动链轮            从动链轮

特点：
- 两轮转向相同
- 传动比精确（不打滑）
- 可传递较大力
```

### 3.2 链条传动特点

| 特点 | 说明 |
|------|------|
| 转向相同 | 两轮同向旋转 |
| 传动比精确 | 不会打滑 |
| 可传大力 | 比皮带强 |
| 需要润滑 | 减少磨损 |

### 3.3 生活实例

| 实例 | 应用 |
|------|------|
| 自行车 | 脚踏板→后轮 |
| 摩托车 | 发动机→后轮 |
| 坦克履带 | 驱动轮→履带 |

### 3.4 教学话术

> "链条就像一串'小手拉小手'，把一个轮子的转动传给另一个轮子。自行车就是用链条把你踩脚踏板的力传到后轮的！"

## 四、传动机构对比

| 传动方式 | 转向 | 传动比 | 距离 | 力量 | 应用 |
|---------|------|--------|------|------|------|
| 齿轮 | 相反 | 精确 | 近 | 大 | 变速箱 |
| 皮带 | 相同 | 不精确 | 远 | 中 | 洗衣机 |
| 链条 | 相同 | 精确 | 中 | 大 | 自行车 |

### 选择原则

```
需要精确传动 → 齿轮或链条
需要远距离传动 → 皮带
需要改变转向 → 齿轮
需要过载保护 → 皮带
需要传递大力 → 齿轮或链条
```

## 五、齿轮组应用

### 5.1 减速齿轮组

```
目的：降低转速，增大力矩

马达 → 小齿轮 → 大齿轮 → 输出
       (快)      (慢)
       (力小)    (力大)

应用：起重机、电动车
```

### 5.2 加速齿轮组

```
目的：提高转速，减小力矩

输入 → 大齿轮 → 小齿轮 → 输出
       (慢)      (快)
       (力大)    (力小)

应用：搅拌机、风扇
```

### 5.3 多级齿轮组

```
目的：获得更大的传动比

输入 → 齿轮1 → 齿轮2 → 齿轮3 → 齿轮4 → 输出
       20齿    40齿    20齿    40齿

总传动比 = (40/20) × (40/20) = 4

应用：钟表、精密仪器
```

## 七、常见错误与禁忌

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| ❌ 齿轮不啮合 | 无法传动 | 确保齿轮牙齿咬合 |
| ❌ 齿轮间隙太大 | 传动不稳 | 调整齿轮距离 |
| ❌ 皮带太松 | 打滑 | 调整皮带张力 |
| ❌ 链条太松 | 跳齿 | 调整链条张力 |

## 八、教学建议

### 8.1 教学顺序

1. **观察实物**：展示齿轮、皮带、链条
2. **动手操作**：让学生转动齿轮
3. **发现规律**：引导观察转向和速度
4. **联系生活**：找出生活中的例子

### 8.2 常见教学难点

| 难点 | 解决方案 |
|------|---------|
| 齿轮转向规律 | 用手指模拟齿轮啮合 |
| 传动比计算 | 用实物数齿数 |
| 皮带vs齿轮转向 | 对比演示 |
