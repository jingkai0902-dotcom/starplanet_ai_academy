## 一、连杆机构

### 1.1 基本概念

连杆机构通过杆件的连接，将一种运动形式转换为另一种运动形式。

```
四连杆机构：

    ┌────────────────┐ 连杆
    │                │
曲柄│                │摇杆
    │                │
    └────────────────┘
          机架

功能：圆周运动 ↔ 往复摆动
```

### 1.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 具体内容 |
|----------|---------|------|---------|
| STEM 3-1-09 | 连杆机构 | 5-6岁 | 曲柄摇杆机构 |
| STEM 2-3-04 | 小鸡啄米 | 4-5岁 | 连杆应用 |

### 1.3 四连杆机构类型

#### 曲柄摇杆机构

```
        连杆
    ┌──────────┐
    │          │
曲柄│          │摇杆
(360°)        (摆动)
    │          │
    └──────────┘
       机架

特点：
- 曲柄可以连续旋转360°
- 摇杆只能在一定角度内摆动
- 将旋转运动转换为摆动

应用：缝纫机、雨刷器
```

#### 双曲柄机构

```
        连杆
    ┌──────────┐
    │          │
曲柄1         曲柄2
(360°)        (360°)
    │          │
    └──────────┘
       机架

特点：两个曲柄都能连续旋转
应用：火车车轮连杆
```

#### 双摇杆机构

```
        连杆
    ┌──────────┐
    │          │
摇杆1         摇杆2
(摆动)        (摆动)
    │          │
    └──────────┘
       机架

特点：两个摇杆都只能摆动
应用：电风扇摇头机构
```

### 1.4 曲柄滑块机构

```
曲柄滑块机构：

    曲柄
      │
      │  连杆
      └────────┐
               │
          ═════╪═════ 滑块（直线运动）
               │
          ─────┴─────

功能：圆周运动 ↔ 直线往复运动
应用：活塞发动机、压缩机
```

### 1.5 教学话术

> "连杆就像'翻译官'，把转圈圈的运动翻译成前后动或上下动。手柄转一圈，小人的腿就迈一步，这就是连杆的魔法！"

### 1.6 生活实例

| 实例 | 机构类型 | 运动转换 |
|------|---------|---------|
| 缝纫机 | 曲柄摇杆 | 旋转→针上下 |
| 雨刷器 | 曲柄摇杆 | 旋转→摆动 |
| 活塞发动机 | 曲柄滑块 | 直线→旋转 |
| 打气筒 | 曲柄滑块 | 直线→压缩 |
| 火车车轮 | 双曲柄 | 同步旋转 |

## 二、凸轮机构

### 2.1 基本概念

凸轮是一个不规则形状的轮子，通过轮廓的变化控制从动件的运动。

```
凸轮机构：

    从动件（上下运动）
        │
        ▼
    ┌───────┐
   /         \
  │     ●     │  ← 凸轮（旋转）
   \         /
    └───────┘

凸轮形状决定从动件的运动规律
```

### 2.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 具体内容 |
|----------|---------|------|---------|
| STEM 3-1-10 | 凸轮机构 | 5-6岁 | 圆心凸轮、心形凸轮 |

### 2.3 凸轮类型

#### 圆心凸轮（偏心轮）

```
圆心凸轮：

      │ 从动件
      ▼
    ┌───┐
   /  ●  \  ← 圆心偏离旋转中心
  │   ○   │  ← 旋转中心
   \     /
    └───┘

运动特点：平稳的正弦运动
应用：平稳升降
```

#### 心形凸轮

```
心形凸轮：

      │ 从动件
      ▼
    ╱╲
   ╱  ╲
  │ ○  │  ← 旋转中心
   ╲  ╱
    ╲╱

运动特点：快速往复
应用：快速敲击
```

#### 盘形凸轮

```
盘形凸轮：

      │ 从动件
      ▼
    ┌─────────┐
   /           \
  │      ○      │
   \     ╱     /
    └───╱─────┘
       ╱ 凸起部分

运动特点：有停顿的间歇运动
应用：自动机械、音乐盒
```

### 2.4 凸轮轮廓与运动关系

```
凸轮轮廓：        从动件运动：

    高点              ↑ 最高位置
     /\              /\
    /  \            /  \
   /    \          /    \
  /      \        /      \
 /        \      /        \
低点       低点  ↓ 最低位置

凸轮转一圈，从动件完成一个升降周期
```

### 2.5 教学话术

> "凸轮就像一个'不规则的轮子'，它转的时候，上面的东西就会一高一低地跳。圆形凸轮像慢悠悠的波浪，心形凸轮像快速的心跳。"

### 2.6 生活实例

| 实例 | 凸轮类型 | 功能 |
|------|---------|------|
| 音乐盒 | 盘形凸轮 | 控制音符 |
| 缝纫机 | 盘形凸轮 | 控制送布 |
| 发动机气门 | 盘形凸轮 | 控制进排气 |
| 自动售货机 | 盘形凸轮 | 控制出货 |

## 三、曲柄机构

### 3.1 基本概念

曲柄是一种能够连续旋转的杆件，常与连杆配合使用。

```
曲柄结构：

    旋转轴
      │
      │
      └────● 曲柄销
           │
           │ 连杆
           │
```

### 3.2 曲柄的作用

```
曲柄的运动：

        ●──→ 曲柄销轨迹（圆形）
       /│\
      / │ \
     /  │  \
    ●───┼───●
     \  │  /
      \ │ /
       \│/
        ●

曲柄销做圆周运动
连杆将圆周运动转换为其他运动
```

### 3.3 曲柄应用

| 应用 | 配合机构 | 功能 |
|------|---------|------|
| 自行车脚踏板 | 链条 | 人力→旋转 |
| 手摇发电机 | 发电机 | 旋转→电能 |
| 绞肉机 | 螺旋 | 旋转→推进 |
| 井轱辘 | 绳索 | 旋转→提升 |

## 四、槽轮机构

### 4.1 基本概念

槽轮机构是一种间歇运动机构，将连续旋转转换为间歇旋转。

```
槽轮机构：

    主动轮（连续转）     槽轮（间歇转）
        ●──────────────────┐
       /│                  │
      / │              ┌───┼───┐
     /  │              │   │   │
    ●───┼───●          │   ●   │
     \  │              │       │
      \ │              └───────┘
       \│
        ●

主动轮转一圈，槽轮转1/4圈（四槽）
```

### 4.2 槽轮机构特点

| 特点 | 说明 |
|------|------|
| 间歇运动 | 转-停-转-停 |
| 定位准确 | 每次转动角度固定 |
| 结构简单 | 零件少 |

### 4.3 生活实例

| 实例 | 功能 |
|------|------|
| 电影放映机 | 间歇送片 |
| 自动售货机 | 间歇出货 |
| 钟表秒针 | 间歇走动 |

## 五、运动转换对比

| 机构 | 输入运动 | 输出运动 | 特点 |
|------|---------|---------|------|
| 曲柄摇杆 | 旋转 | 摆动 | 连续 |
| 曲柄滑块 | 旋转 | 直线往复 | 连续 |
| 凸轮 | 旋转 | 任意规律 | 可设计 |
| 槽轮 | 旋转 | 间歇旋转 | 间歇 |

### 运动转换选择

```
需要摆动输出 → 曲柄摇杆
需要直线往复 → 曲柄滑块
需要特定规律 → 凸轮
需要间歇运动 → 槽轮
```

## 七、常见错误与禁忌

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| ❌ 连杆长度不对 | 卡死 | 按设计图纸搭建 |
| ❌ 凸轮轮廓不光滑 | 运动不平稳 | 打磨光滑 |
| ❌ 曲柄销松动 | 脱落 | 确保紧固 |
| ❌ 槽轮间隙太大 | 定位不准 | 调整间隙 |

## 八、教学建议

### 8.1 教学顺序

1. **观察现象**：展示运动转换效果
2. **分析结构**：拆解机构组成
3. **动手搭建**：按图纸搭建
4. **创意设计**：设计自己的运动规律

### 8.2 常见教学难点

| 难点 | 解决方案 |
|------|---------|
| 四连杆类型区分 | 用"哪个能转360°"判断 |
| 凸轮运动规律 | 用动画或慢动作演示 |
| 曲柄滑块理解 | 用活塞发动机模型 |
