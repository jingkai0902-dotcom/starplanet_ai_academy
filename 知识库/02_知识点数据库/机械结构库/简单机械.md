## 一、杠杆

### 1.1 基本概念

杠杆是最基本的简单机械，由一根硬棒和一个支点组成。

```
杠杆三要素：

    力点(F)           支点(O)           重点(W)
      ↓                 △                 ↓
  ════════════════════════════════════════════
      ←────��� 力臂 ─────→←───── 阻力臂 ─────→

杠杆原理：力 × 力臂 = 阻力 × 阻力臂
```

### 1.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 杠杆类型 | 具体内容 |
|----------|---------|------|---------|---------|
| STEM 2-1-03 | 快乐跷跷板 | 4-5岁 | 等臂杠杆 | 平衡原理 |
| STEM 2-1-04 | 神奇的投石机 | 4-5岁 | 费力杠杆 | 省距离 |
| STEM 3-1-01 | 杠杆与平衡 | 5-6岁 | 省力杠杆 | 三要素 |
| STEM 3-2-06 | 古代投石机 | 5-6岁 | 费力杠杆 | 弹性势能 |

### 1.3 杠杆分类

#### 省力杠杆（第一类）

```
支点在中间，力臂 > 阻力臂

    F↓              W↓
    ════════△════════
    长力臂    短阻力臂

特点：省力，但费距离
实例：撬棍、开瓶器、剪刀
```

#### 费力杠杆（第二类）

```
支点在一端，力臂 < 阻力臂

    F↓                    W↓
    △════════════════════════
    短力臂        长阻力臂

特点：费力，但省距离（速度快）
实例：筷子、钓鱼竿、投石机
```

#### 等臂杠杆（第三类）

```
支点在中间，力臂 = 阻力臂

    F↓              W↓
    ════════△════════
    力臂  =  阻力臂

特点：不省力也不费力
实例：天平、跷跷板
```

### 1.4 教学话术

**中班（4-5岁）**：
> "跷跷板就像一个大杠杆。中间的支撑点叫'支点'，就像跷跷板的'腰'。你坐的地方叫'力点'，小伙伴坐的地方叫'重点'。"

**大班（5-6岁）**：
> "如果你想用很小的力气抬起很重的东西，应该把支点靠近谁？对了，靠近重物！这样你的力臂就变长了，就更省力。"

### 1.5 实验活动

#### 跷跷板平衡实验

```
材料：积木、尺子、小物品

步骤：
1. 用积木做支点，尺子做杠杆
2. 在两端放不同重量的物品
3. 调整位置使其平衡
4. 记录：重的物品要靠近支点

结论：重量 × 距离 = 重量 × 距离
```

#### 投石机实验

```
材料：投石机模型、小球

步骤：
1. 搭建投石机
2. 改变力臂长度
3. 测量投掷距离
4. 记录数据

结论：力臂越短，投掷越远（但需要更大的力）
```

## 二、滑轮

### 2.1 基本概念

滑轮是一个边缘有槽的轮子，绳子绕在槽里，可以改变力的方向或省力。

### 2.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 滑轮类型 | 具体内容 |
|----------|---------|------|---------|---------|
| STEM 2-1-07 | 有趣的滑轮 | 4-5岁 | 定滑轮 | 改变方向 |
| STEM 3-1-04 | 滑轮系统 | 5-6岁 | 滑轮组 | 省力原理 |

### 2.3 滑轮分类

#### 定滑轮

```
      ┌─────┐
      │  ○  │ ← 固定在支架上
      └──┬──┘
         │
    ↓拉  │  ↑升
         │
        ███ 重物

特点：
- 轴固定不动
- 改变力的方向
- 不省力（拉力 = 重力）
- 方便操作（向下拉比向上推容易）
```

#### 动滑轮

```
         │
         │ ← 绳子一端固定
      ┌──┴──┐
      │  ○  │ ← 随重物一起移动
      └──┬──┘
         │
        ███ 重物

特点：
- 轴随重物移动
- 不改变力的方向
- 省力一半（拉力 = 重力/2）
- 但要拉两倍的距离
```

#### 滑轮组

```
      ┌─────┐ 定滑轮
      │  ○  │
      └──┬──┘
         │
      ┌──┴──┐ 动滑轮
      │  ○  │
      └──┬──┘
         │
        ███

特点：
- 结合定滑轮和动滑轮
- 既改变方向又省力
- 绳子段数越多越省力
```

### 2.4 教学话术

**中班（4-5岁）**：
> "固定不动的滑轮，像个'方向盘'，你往下拉，东西往上升。虽然不省力，但是站在地上往下拉比爬上去往上推方便多了！"

**大班（5-6岁）**：
> "跟着物体一起动的滑轮，像个'小帮手'，帮你分担一半的力气。但是你要拉两倍的绳子长度，这就是'省力必费距离'。"

### 2.5 实验活动

#### 滑轮省力实验

```
材料：滑轮、绳子、弹簧秤、重物

步骤：
1. 直接用弹簧秤提起重物，记录读数
2. 用定滑轮提起，记录读数
3. 用动滑轮提起，记录读数
4. 比较三次读数

结论：
- 直接提：读数 = 重力
- 定滑轮：读数 = 重力（不省力）
- 动滑轮：读数 ≈ 重力/2（省力一半）
```

## 三、轮轴

### 3.1 基本概念

轮轴是由一个大轮和一个小轴组成的简单机械，本质上是一个连续旋转的杠杆。

```
轮轴结构：

      ┌─────────────┐
      │             │
      │    轮       │  ← 大轮（相当于长力臂）
      │  ┌─────┐    │
      │  │ 轴  │    │  ← 小轴（相当于短力臂）
      │  └─────┘    │
      │             │
      └─────────────┘

轮轴 = 旋转的杠杆
```

### 3.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 具体内容 |
|----------|---------|------|---------|
| STEM 3-1-03 | 轮轴原理 | 5-6岁 | 井轱辘、方向盘 |

### 3.3 轮轴原理

```
省力轮轴（轮大轴小）：

在轮上用力 → 轴上输出
   小力         大力

实例：方向盘、门把手、水龙头

费力轮轴（轮小轴大）：

在轴上用力 → 轮上输出
   大力         小力（但速度快）

实例：自行车脚踏板
```

### 3.4 教学话术

> "轮轴就像一个'旋转的杠杆'。轮是长力臂，轴是短力臂。汽车方向盘为什么那么大？因为大轮子转起来更轻松！"

### 3.5 生活实例

| 实例 | 轮 | 轴 | 作用 |
|------|----|----|------|
| 方向盘 | 方向盘 | 转向柱 | 省力转向 |
| 门把手 | 把手 | 门锁芯 | 省力开门 |
| 水龙头 | 旋钮 | 阀芯 | 省力开关 |
| 井轱辘 | 摇把 | 轴 | 省力提水 |
| 螺丝刀 | 手柄 | 刀杆 | 省力拧螺丝 |

## 四、斜面与螺旋

### 4.1 基本概念

斜面是一个倾斜的平面，可以用较小的力把物体提升到较高的位置。

```
斜面：

        /│
       / │
      /  │ 高度(h)
     /   │
    /    │
   /─────┘
    长度(L)

斜面越长（坡度越缓），越省力
但要走更长的路
```

### 4.2 跨课程出现

| 课程编号 | 课程名称 | 年龄 | 具体内容 |
|----------|---------|------|---------|
| STEM 3-1-05 | 斜面与螺旋 | 5-6岁 | 盘山公路、螺丝 |

### 4.3 螺旋 = 卷起来的斜面

```
斜面展开：
    ╱╲
   ╱  ╲
  ╱    ╲
 ╱      ╲
╱────────╲

卷成螺旋：
    ╱╲
   ╱  ╲
  ╱    ╲
 ╱      ╲
╱────────╲
   ↓ 卷起来
  ┌───┐
  │╱╲ │
  │╱╲ │
  │╱╲ │
  └───┘
  螺丝钉
```

### 4.4 教学话术

> "斜面就像'懒人的楼梯'，虽然走得远，但是走得轻松。盘山公路为什么要绕来绕去？因为直接爬上去太陡了！"

> "螺丝钉就是把斜面卷成了一圈一圈的，每转一圈就往前走一点点。螺纹越密（斜面越长），拧起来越轻松。"

### 4.5 生活实例

| 实例 | 类型 | 作用 |
|------|------|------|
| 楼梯 | 斜面 | 省力上楼 |
| 盘山公路 | 斜面 | 省力上山 |
| 滑梯 | 斜面 | 利用重力下滑 |
| 螺丝钉 | 螺旋 | 省力固定 |
| 瓶盖 | 螺旋 | 省力密封 |
| 千斤顶 | 螺旋 | 省力顶起汽车 |

## 五、简单机械对比

| 机械 | 省力原理 | 代价 | 典型应用 |
|------|---------|------|---------|
| 杠杆 | 力臂长 | 移动距离大 | 撬棍、跷跷板 |
| 滑轮 | 绳子段数多 | 拉绳距离长 | 起重机、电梯 |
| 轮轴 | 轮大轴小 | 转动圈数多 | 方向盘、门把手 |
| 斜面 | 坡度缓 | 行走距离长 | 楼梯、盘山公路 |

**共同规律**：省力必费距离，功的总量不变

## 七、常见错误与禁忌

| 错误 | 后果 | 正确做法 |
|------|------|---------|
| ❌ 杠杆支点不稳 | 无法平衡 | 固定支点位置 |
| ❌ 滑轮绳子脱槽 | 无法提升 | 绳子放在槽内 |
| ❌ 轮轴松动 | 打滑 | 确保轮轴紧固 |
| ❌ 斜面太陡 | 不省力 | 降低坡度 |

## 八、教学建议

### 8.1 教学顺序

1. **先体验**：让学生亲手操作，感受省力效果
2. **再观察**：引导观察结构特点
3. **后总结**：归纳原理，联系生活

### 8.2 常见教学难点

| 难点 | 解决方案 |
|------|---------|
| 杠杆三要素混淆 | 用"腰、手、重物"类比 |
| 滑轮省力原理 | 实际称重对比 |
| 轮轴理解困难 | 用方向盘实物演示 |
| 螺旋与斜面关系 | 用纸条卷成螺旋演示 |
