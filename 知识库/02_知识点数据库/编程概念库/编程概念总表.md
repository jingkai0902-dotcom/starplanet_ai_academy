```
CODE1-3-05          CODE1-3-07          PYAI 1-0-5          PYAI 1-0-6          PYAI 1-3-5

(死循环)            (条件循环)          (无限循环)          (计数循环)           (遍历循环)
```

## 教学建议

### 概念引入顺序

1. **顺序执行** → 理解程序从上到下执行
2. **循环** → 理解重复的力量
3. **条件判断** → 理解程序的"选择"
4. **变量** → 理解数据存储
5. **函数** → 理解代码复用
6. **列表** → 理解批量数据
7. **OOP** → 理解对象思维

### 常见教学难点

| 概念 | 难点 | 教学建议 |
|------|------|---------|
| 循环 | 理解"重复"的抽象 | 用生活例子（刷牙10次） |
| 条件 | 理解"判断"的逻辑 | 用选择题类比 |
| 变量 | 理解"存储"的概念 | 用盒子比喻 |
| 函数 | 理解"封装"的意义 | 用工具箱比喻 |
| 递归 | 理解"自我调用" | 用俄罗斯套娃比喻 |
