### 1.3 Scratch角度系统

```
        0°
        ↑
   270° ← → 90°
        ↓
       180°
```

**特点**：
- 0°朝上（与数学不同！）
- 顺时针为正方向
- 范围：-180° 到 180°

### 1.4 常用积木

## 二、多边形与内角

### 2.1 外角公式

```
正n边形的外角 = 360° / n
```

| 多边形 | 边数 | 外角 |
|--------|------|------|
| 正三角形 | 3 | 120° |
| 正方形 | 4 | 90° |
| 正五边形 | 5 | 72° |
| 正六边形 | 6 | 60° |

### 2.2 跨课程出现

| 课程编号 | 课程名称 | 具体内容 |

| PYAI 1-1-4 | 对照实验 | 多参数多边形 |

### 2.3 Python绘制多边形

```python
import turtle

def draw_polygon(sides, length):
    """绘制正多边形"""
    angle = 360 / sides  # 外角
    for i in range(sides):
        turtle.forward(length)
        turtle.right(angle)

# 绘制正六边形
draw_polygon(6, 50)
```

## 三、角度与弧度转换

### 3.1 转换公式

```
弧度 = 角度 × π / 180
角度 = 弧度 × 180 / π
```

| 角度 | 弧度 |
|------|------|
| 0° | 0 |
| 30° | π/6 |
| 45° | π/4 |
| 60° | π/3 |
| 90° | π/2 |
| 180° | π |
| 360° | 2π |

### 3.2 跨课程出现

| 课程编号 | 课程名称 | 具体内容 |
|----------|---------|---------|
| PYAI 2-1-2 | 笔走龙蛇 | math.radians() |
| CODE3-2-06 | 篮球小子 | 抛物线角度 |

### 3.3 Python转换函数

```python
import math

# 角度转弧度
angle_deg = 90
angle_rad = math.radians(angle_deg)  # 1.5707... (π/2)

# 弧度转角度
angle_rad = math.pi / 4
angle_deg = math.degrees(angle_rad)  # 45.0

# 注意：Python三角函数使用弧度！
sin_30 = math.sin(math.radians(30))  # 0.5
cos_60 = math.cos(math.radians(60))  # 0.5
```

### 3.4 教学话术

> "Python的三角函数用的是弧度，不是角度。所以我们要先用math.radians()把角度转成弧度。记住：180度等于π弧度。"

## 四、三角函数

### 4.1 基本定义

```
        /|
       / |
    斜/  | 对边
  边/   |
   /θ   |
  /_____|
   邻边

sin(θ) = 对边 / 斜边
cos(θ) = 邻边 / 斜边
tan(θ) = 对边 / 邻边
```

### 4.2 常用值

| 角度 | sin | cos | tan |
|------|-----|-----|-----|
| 0° | 0 | 1 | 0 |
| 30° | 0.5 | 0.866 | 0.577 |
| 45° | 0.707 | 0.707 | 1 |
| 60° | 0.866 | 0.5 | 1.732 |
| 90° | 1 | 0 | ∞ |

| 课程编号 | 课程名称 | 具体内容 |
|----------|---------|---------|
| PYAI 2-1-2 | 笔走龙蛇 | 绘制正弦曲线 |
| CODE3-2-06 | 篮球小子 | 抛物线速度分解 |

### 4.4 绘制正弦曲线

```python
import turtle
import math

t = turtle.Turtle()
t.speed(0)
t.penup()
t.goto(-200, 0)
t.pendown()

# 绘制正弦曲线
for x in range(-200, 201):
    rad = math.radians(x * 2)  # 放大周期
    y = math.sin(rad) * 100    # 放大振幅
    t.goto(x, y)

turtle.done()
```

### 4.5 速度分解

```python
import math

# 已知：速度大小和角度
speed = 50
angle = 45  # 度

# 分解为水平和垂直分量
vx = speed * math.cos(math.radians(angle))  # 水平速度
vy = speed * math.sin(math.radians(angle))  # 垂直速度

print(f"水平速度: {vx:.2f}")  # 35.36
print(f"垂直速度: {vy:.2f}")  # 35.36
```

## 五、圆周运动

### 5.1 基本公式

```python
# 圆周运动坐标
x = center_x + radius * cos(angle)
y = center_y + radius * sin(angle)
```

### 5.2 跨课程出现

| 课程编号 | 课程名称 | 具体内容 |
|----------|---------|---------|
| CODE1-4-08 | 万花筒 | 旋转图案 |
| PYAI 2-2-7 | 积小致巨1 | 粒子圆周运动 |

### 5.3 Python圆周运动

```python
import pygame
import math

pygame.init()
screen = pygame.display.set_mode((400, 400))
clock = pygame.time.Clock()

center_x, center_y = 200, 200
radius = 100
angle = 0

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    screen.fill((0, 0, 0))

    # 计算圆周上的位置
    x = center_x + radius * math.cos(math.radians(angle))
    y = center_y + radius * math.sin(math.radians(angle))

    pygame.draw.circle(screen, (255, 0, 0), (int(x), int(y)), 10)

    angle += 2  # 每帧旋转2度

    pygame.display.flip()
    clock.tick(60)

pygame.quit()
```

### 6.2 常见教学难点

| 难点 | 解决方案 |
|------|---------|
| Scratch方向与数学不同 | 强调"0度朝上"，用指南针类比 |

### 6.3 禁忌提醒

| 禁忌 | 后果 | 正确做法 |
|------|------|---------|
| ❌ 直接用角度调用sin/cos | 结果错误 | 先用radians()转换 |
